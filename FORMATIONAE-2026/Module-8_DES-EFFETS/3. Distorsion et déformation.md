# Warp — la déformation libre

**Effect → Distort → Warp**

Warp applique une **déformation géométrique globale** à un calque selon des formes prédéfinies — arc, courbe, drapeau, poisson, gonflement, torsion…

**Les paramètres :**

```
Warp Style  → type de déformation (Arc, Arc Lower, Arc Upper, Arch,
               Bulge, Shell Lower, Shell Upper, Flag, Wave,
               Fish, Rise, Fisheye, Inflate, Squeeze, Twist)
Bend        → intensité de la déformation (-100 à +100)
              valeur négative = déformation inverse
Horizontal Distortion → déformation asymétrique sur l'axe X
Vertical Distortion   → déformation asymétrique sur l'axe Y
```

**Usage typique — texte courbé en arc :**

```
Warp Style          = Arc
Bend                = 30
Horizontal Distortion = 0
Vertical Distortion   = 0
```

> 💡 Warp est parfait pour des déformations simples et propres sur du texte ou des formes. Pour des déformations plus complexes et libres, passe à Mesh Warp.

**Warp animé — le drapeau qui flotte :**

```
Warp Style = Flag
Bend       = 30 (fixe)
```

Ajoute ensuite un **Noise** ou un **Turbulent Displace** avec une faible valeur, animé en `time * 50` sur l'Evolution pour simuler le mouvement du tissu.

# Mesh Warp — le contrôle point par point

**Effect → Distort → Mesh Warp**

Mesh Warp divise le calque en une **grille de points de contrôle** déplaçables individuellement. C'est la déformation la plus précise disponible nativement dans AE.

**Les paramètres :**

```
Rows    → nombre de rangées de la grille (plus = plus de précision)
Columns → nombre de colonnes
Quality → précision du calcul de déformation
```

**Utilisation :**

1. Appliquer l'effet → une grille apparaît sur le calque dans le panneau Composition
2. Sélectionner l'outil **Direct Selection** (`A`)
3. Cliquer et déplacer les points de la grille directement dans le panneau Comp

Chaque point peut être animé par keyframe — ce qui permet de morphe une forme progressivement.

**Usage typique — adapter un logo à une surface courbée :**

Un logo doit s'appliquer sur la carrosserie courbée d'une voiture filmée. Mesh Warp permet de plier le logo pour qu'il suive les courbes de la surface, point par point.

> 💡 Mesh Warp ne remplace pas les plugins de tracking et match move, mais il est utile pour des compositions rapides où la précision absolue n'est pas requise.

# Liquify — la déformation interactive

**Effect → Distort → Liquify**

Liquify est un effet de **peinture de déformation** — on "peint" directement sur le calque avec des outils qui poussent, tirent, tourbillonnent, gonflent ou rétractent les pixels.

**Les outils disponibles :**

```
Warp      → pousse les pixels dans la direction du pinceau
Turbulence → mélange les pixels de manière chaotique
Twirl Clockwise / Counter-Clockwise → tourbillon
Pucker    → aspire les pixels vers le centre du pinceau
Bloat     → gonfle les pixels vers l'extérieur du pinceau
Shift Pixels → déplace les pixels perpendiculairement
Reflection → crée un effet miroir
```

**Les paramètres du pinceau :**

```
Brush Size     → taille du pinceau
Brush Pressure → intensité de l'effet (1–100)
Turbulent Jitter → aléatoire du Turbulence
```

**Déformation animée :**

Liquify stocke toutes les déformations dans un **Distortion Mesh** interne. Ce mesh peut être animé frame par frame en activant le **keyframe sur Distortion Mesh** — mais c'est fastidieux. Pour des animations fluides, préfère Turbulent Displace.

> 💡 Liquify est redoutable pour **corriger des éléments** : aplatir un reflet non désiré sur une surface, ajuster subtilement une forme qui ne colle pas à un autre élément. Il est moins utilisé pour des animations que pour du polish de composition.

# Displacement Map — déformer par une image

**Effect → Distort → Displacement Map**

Displacement Map déplace les pixels d'un calque **en lisant les valeurs d'un autre calque** (ou du même calque). Les zones blanches de la map déplacent fortement, les zones noires peu ou pas, les zones grises moyennement.

**Les paramètres :**

```
Displacement Map Layer → le calque qui sert de map (peut être caché)
Use For Horizontal Displacement → quelle couche de couleur pilote X
                                   Red / Green / Blue / Alpha / Luminance / Hue
Use For Vertical Displacement   → idem pour Y
Max Horizontal Displacement → amplitude maximale du déplacement en X (en pixels)
Max Vertical Displacement   → idem en Y
Displacement Map Behavior   → Center Map / Stretch Map to Fit / Tile Map
Edge Behavior               → Wrap / Smear / Edge pixels
```

**Usage typique — texte qui tremble comme de la chaleur :**

1. Créer un calque de bruit : **Layer → New → Solid** → appliquer **Fractal Noise** dessus
2. Animer l'Evolution du Fractal Noise sur `time * 100` (expression)
3. Rendre le calque de bruit invisible (œil éteint)
4. Sur le calque texte : **Displacement Map** → sélectionner le calque de bruit
5. Régler Max Horizontal et Vertical Displacement à 15–20px

Le texte ondule comme de l'air chaud au-dessus d'un asphalte.

**Usage typique — reflet dans l'eau :**

Le même principe avec un Fractal Noise plus lent (Softness élevé, Evolution lente) et des valeurs de déplacement plus importantes sur l'axe vertical.

> 💡 Le Displacement Map peut lire le **calque lui-même** (Displacement Map Layer = None → utilise le calque courant). Cela crée des auto-déformations — utile pour des effets de liquide ou de plasma.

# Turbulent Displace — déformation organique procédurale

**Effect → Distort → Turbulent Displace**

Turbulent Displace est la version **animée et procédurale** du déplacement. Il génère sa propre map de bruit interne et la fait évoluer dans le temps — sans calque source séparé.

**Les paramètres :**

```
Displacement → type de déformation
               Turbulent / Turbulent Smoother / Bulge / Bulge Smoother /
               Twist / Vertical / Horizontal / Cross
Amount       → amplitude de la déformation en pixels
Size         → taille des "vagues" du bruit
               petit = nombreuses petites déformations
               grand = quelques grandes déformations
Offset       → décale la position du bruit dans l'espace
Complexity   → nombre de couches de bruit superposées (1–10)
               plus élevé = plus organique, plus coûteux
Evolution    → la variable qui fait "avancer" le bruit dans le temps
               à relier à time : expression → time * 50
Pinning      → fixe certains bords du calque (pour les drapeaux, bannières)
               Pin All Edges / Pin Left / Pin Right / Pin Top / Pin Bottom
```

**La clé : l'Evolution**

Sans Evolution, Turbulent Displace produit une déformation statique. L'animation vient entièrement de cette valeur :

```javascript
// Sur le paramètre Evolution :
time * 80
// Vitesse modérée — ajuster le multiplicateur selon le besoin
// Valeur faible (20–40) → lent, organique
// Valeur haute (150+)  → rapide, chaotique
```

**Usage typique — drapeau animé :**

```
Displacement = Turbulent
Amount       = 20
Size         = 80
Complexity   = 2
Evolution    → time * 40 (expression)
Pinning      = Pin Left (le bord gauche reste fixe, comme accroché à un mât)
```

**Usage typique — chaleur, distorsion atmosphérique :**

```
Displacement = Turbulent Smoother
Amount       = 8
Size         = 200
Complexity   = 3
Evolution    → time * 60
```

> 💡 Turbulent Displace est l'un des effets les plus polyvalents d'AE. Partout où tu as besoin d'un mouvement organique sur un calque — tissu, eau, chaleur, magie, explosion — il est la première option à considérer avant de chercher un plugin.

# CC Lens — déformation sphérique

**Effect → Distort → CC Lens**

CC Lens applique une **déformation sphérique** sur le calque — comme si l'image était projetée sur une boule de verre ou regardée à travers un objectif grand-angle.

**Les paramètres :**

```
Center     → centre de la déformation
Size       → rayon de la sphère en % du calque
Convergence → intensité et direction
               positif  → boule convexe (loupe)
               négatif  → boule concave (bol)
               extrême  → effet "fish eye" ou "anti-fish eye"
```

**Usage typique — globe, planète, interface holographique :**

```
Size        = 100
Convergence = 60   → donne un aspect bombé à l'image entière
```

**Usage typique — l'œil déformé derrière un verre :**

```
Center      = sur l'iris de l'œil filmé
Size        = 30
Convergence = 80
```

> 💡 CC Lens animé sur Convergence de 0 à 100 crée un effet de "zoom sphérique" cinématographique très différent d'un zoom optique classique. Souvent utilisé dans les génériques et les transitions.

# Optics Compensation — corriger et exagérer l'optique

**Effect → Distort → Optics Compensation**

Optics Compensation corrige ou exagère la **distorsion optique d'un objectif** — principalement la distorsion en barillet (fish eye) et la distorsion en coussinet (anti-fish eye).

**Le paramètre principal :**

```
Field of View (FOV) → angle de vue simulé en degrés
                       valeur positive → corrige le fish eye
                       valeur négative → l'exagère
Reverse Lens Distortion → inverse la direction de la correction
Optimal Pixels         → maximise le nombre de pixels nets
Fit in Comp            → recadre le résultat pour remplir le cadre
```

**Usage 1 — corriger un fish eye :**

Un footage filmé avec un GoPro ou un grand-angle extrême présente une distorsion en barillet très marquée. Optics Compensation avec FOV positif + Reverse Lens Distortion corrige cette distorsion pour rendre l'image plus "droite".

```
Field of View           = 75   (à ajuster selon l'objectif source)
Reverse Lens Distortion = ✅
Fit in Comp             = ✅
```

**Usage 2 — créer un effet fish eye sur n'importe quel calque :**

```
Field of View           = 100
Reverse Lens Distortion = ❌
```

> 💡 Optics Compensation et CC Lens produisent des effets proches visuellement, mais avec des résultats différents aux bords. Optics Compensation est plus adapté pour corriger un vrai footage optique ; CC Lens est plus adapté pour des effets stylisés sur des calques synthétiques.
