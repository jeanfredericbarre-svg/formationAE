# CC Particle World — le générateur de particules natif

**Effect → Simulation → CC Particle World**

CC Particle World est le simulateur de particules 3D natif d'AE. Il génère des particules dans un espace 3D (visible depuis la caméra AE si le calque est en 3D), avec des paramètres de physique, de forme et de durée de vie.

> 💡 Pour des systèmes de particules vraiment complexes (pluie dense, explosions, foule), des plugins comme Particular de Red Giant sont la référence professionnelle. CC Particle World couvre les besoins courants du motion design sans plugin.

**Les paramètres essentiels :**

### Birth Rate et Longevity
```
Birth Rate  → nombre de particules émises par frame
              valeur haute = nuage dense / valeur faible = particules éparses
Longevity   → durée de vie de chaque particule en secondes
              valeur haute = particules qui restent longtemps à l'écran
```

### Producer — la source d'émission
```
Position X / Y / Z  → centre de l'émetteur dans l'espace 3D
Radius X / Y / Z    → taille de la zone d'émission
                      0 = émission depuis un point / 1 = émission volumique
```

### Physics — la dynamique
```
Velocity         → vitesse initiale des particules à l'émission
Gravity          → gravité (positif = vers le bas, négatif = vers le haut)
                   0 = particules sans gravité (utile pour les étoiles, poussières)
Resistance       → résistance de l'air (ralentit les particules progressivement)
Extra            → turbulence chaotique ajoutée au mouvement
Extra Angle      → direction de la turbulence
```

### Particle — l'apparence
```
Particle Type  → forme de chaque particule
                 Line, Star, Faded Sphere, Lens, Shaded Sphere,
                 Bubble, Cloudlet, Textured Disc, Textured Square
                 → Textured Disc / Square : utilise un calque AE comme texture
Birth Size     → taille à l'émission
Death Size     → taille à la mort (si > Birth Size : particule qui grandit)
Birth Color    → couleur à l'émission
Death Color    → couleur à la mort (transition de couleur sur la durée de vie)
Max Opacity    → transparence maximale
```

**Usage typique — pluie :**

```
Birth Rate     = 5.0
Longevity      = 2.0
Producer Radius X = 2 / Radius Y = 0.1 / Radius Z = 1
Velocity       = 1.5
Gravity        = 0.8
Particle Type  = Line
Birth Size     = 0.05
Death Size     = 0.05
Birth Color    = blanc
Extra          = 0.1   ← légère turbulence pour éviter la régularité parfaite
```

**Usage typique — poussière en suspension :**

```
Birth Rate     = 1.5
Longevity      = 4.0
Velocity       = 0.1
Gravity        = -0.05  ← légèrement négatif : les poussières montent doucement
Resistance     = 1.0
Particle Type  = Faded Sphere
Extra          = 0.3
```

**Particules texturées :**

Pour que chaque particule soit un calque AE (logo, icône, symbole) :

1. Créer un calque avec l'élément voulu (solide + shape, texte, PNG)
2. Dans CC Particle World → Particle Type = **Textured Disc** ou **Textured Square**
3. Un nouveau paramètre **Texture Layer** apparaît → sélectionner le calque source
4. Rendre le calque source invisible (œil éteint) — il sert uniquement de texture

# Fractal Noise — le bruit de référence

**Effect → Noise & Grain → Fractal Noise**

Fractal Noise génère une **texture de bruit procédural** — une image en niveaux de gris avec des structures organiques à plusieurs échelles. C'est l'un des effets les plus polyvalents d'AE, utilisé directement comme texture ou comme source pour d'autres effets.

**Les paramètres essentiels :**

```
Fractal Type  → type de bruit
                Basic, Turbulent Smooth, Turbulent Sharp, Rocky,
                Threads, Swirly, Strings...
Noise Type    → algo de génération du bruit
                Block, Linear, Soft Linear, Spline
                → Spline = le plus doux et le plus naturel
Contrast      → contraste du bruit (augmente la différence noir/blanc)
Brightness    → luminosité globale
Overflow      → comportement quand les valeurs dépassent les limites
                Allow HDR Results / Clip / Soft Clamp / Wrap Back and Forth
```

**Transform — contrôle de l'espace :**
```
Scale         → taille des structures du bruit
                grand = grandes formes douces / petit = bruit fin
Offset Turbulence → décale le bruit dans l'espace (pour l'animer)
Rotation      → rotation du bruit
```

**Sub Settings — les couches de bruit :**
```
Sub Influence → influence des couches de détail
Sub Scale     → taille des couches de détail
Complexity    → nombre de couches superposées (1–10)
               plus élevé = plus de détail, plus organique, plus lent
```

**Evolution — l'animation :**
```
Evolution         → la variable qui fait "bouger" le bruit
                    expression : time * 50 (vitesse à ajuster)
Evolution Options
  Random Seed     → graine aléatoire — change la forme du bruit sans changer les réglages
  Cycle Evolution → boucle le bruit sur un nombre de tours défini
                    (pour des boucles parfaites de bruit animé)
```

**Usages du Fractal Noise :**

- **Texture de fond** : posé directement, avec des couleurs ajustées par Curves ou Tritone
- **Map de déplacement** : source pour Displacement Map ou Turbulent Displace
- **Masque organique** : utilisé comme Luma Matte pour des révélations organiques
- **Simulation de surface** : eau, brume, nuages, fumée statique
- **Générateur de boucles** : avec Cycle Evolution, crée un bruit qui boucle parfaitement

# 4-Color Gradient — le fond dégradé 4 points

**Effect → Generate → 4-Color Gradient**

4-Color Gradient génère un dégradé entre **quatre couleurs positionnables librement** dans l'espace du calque. C'est l'outil de référence pour des fonds de composition riches et non-symétriques.

```
Point 1 / 2 / 3 / 4  → positions des quatre sources de couleur
Color 1 / 2 / 3 / 4  → couleurs correspondantes
Blend                 → adoucissement de la transition entre les couleurs
Jitter                → ajoute du bruit aux bords pour éviter le banding
                        (dégradés en niveaux de gris sur fond sombre)
Opacity               → opacité de l'effet
Blending Mode         → mode de fusion de l'effet avec le calque sous-jacent
```

**Usage typique — fond de composition dynamique :**

Placer 4-Color Gradient sur un solide noir en taille composition. Positionner les 4 points dans les coins ou selon la composition. Animer lentement les positions des points pour un fond vivant.

```javascript
// Expression sur Point 1 (légère oscillation autour de sa position) :
var base = [200, 150];
var w = wiggle(0.3, 30);
[base[0] + w[0], base[1] + w[1]]
// Le point oscille doucement — fond ambient qui respire
```

# Stroke — tracer un tracé animé

**Effect → Generate → Stroke**

Stroke trace le **contour d'un masque** sur un calque, avec des paramètres de couleur, d'épaisseur, d'opacité — et surtout, des paramètres d'animation du tracé (Start/End) qui doublent l'effet Trim Paths des Shape Layers, mais applicables à n'importe quel calque.

```
All Masks / Mask      → tracé tous les masques ou un seul masque précis
Stroke Sequentially   → si plusieurs masques, les trace l'un après l'autre
Color                 → couleur du trait
Brush Size            → épaisseur en pixels
Brush Hardness        → dureté des bords (0% = très doux, 100% = bord net)
Opacity               → opacité du trait
Start                 → point de départ du trait sur le tracé (0–100%)
End                   → point d'arrivée du trait (0–100%)
Spacing               → espacement si le trait est en pointillé
Paint Style           → On Original Image / On Transparent / Reveal Original
```

**Start et End animés — l'auto-tracé :**

C'est la fonctionnalité principale de Stroke pour le motion design. Animer **End de 0% à 100%** trace le contour progressivement, exactement comme Trim Paths.

La différence avec Trim Paths : Stroke s'applique sur n'importe quel calque avec un masque — une image importée, un solide, un footage — sans avoir besoin d'un Shape Layer.

**Paint Style = Reveal Original :**

Mode particulier où Stroke ne peint pas une couleur, mais **révèle les pixels du calque** sous-jacent en suivant le tracé. Animé, cela donne l'impression que l'image se dessine elle-même progressivement.

# Vegas — le tracé lumineux automatique

**Effect → Generate → Vegas**

Vegas trace des **segments lumineux** qui courent automatiquement le long des contours d'un calque ou d'un masque, sans avoir à définir Start/End. C'est un effet d'animation de tracé plus automatisé que Stroke.

```
Stroke → Edge Selection   → détecte automatiquement les contours du calque
         Mask/Path         → utilise un masque spécifique
Image Contours → utilise la luminance du calque pour détecter les contours

Segments      → nombre de segments lumineux qui courent sur le tracé
Length        → longueur de chaque segment (0–1 en fraction du tracé total)
Segment Distribution → Evenly Spaced / Random / Random Follow
                        → Evenly Spaced : segments régulièrement espacés
Rotation      → vitesse de déplacement des segments
Width         → épaisseur du trait
Hardness      → dureté des bords
Start Color / End Color → couleurs au début et à la fin de chaque segment
Inside Color  → couleur centrale du segment
Blend Mode    → mode de fusion avec le calque
```

**Usage typique — contour lumineux animé sur un logo :**

```
Stroke          = Mask/Path (masque en forme du logo)
Segments        = 1
Length          = 0.15
Rotation        → time * 0.5 (expression — déplacement continu)
Width           = 3
Blend Mode      = Screen
Start Color     = transparent
End Color       = transparent
Inside Color    = blanc brillant
```

Le résultat : un point de lumière qui court en permanence le long du contour du logo.

# Radio Waves — les ondes procédurales

**Effect → Generate → Radio Waves**

Radio Waves génère des **ondes concentriques** qui se propagent depuis un point d'origine — cercles, polygones, étoiles — avec contrôle de la vitesse de propagation, de l'épaisseur et de la transparence.

```
Producer Point    → origine des ondes
Render Quality    → qualité du rendu (Low/Medium/High)
Wave Type         → forme de l'onde
                    Polygon / Image Contours / Mask
Frequency         → nombre d'ondes émises par seconde
Expansion         → vitesse de propagation
Lifespan          → durée de vie de chaque onde en secondes
Reflection        → réfléchit les ondes aux bords du calque

Wave Motion
  Spin            → rotation de chaque onde sur elle-même
  Wobble Amount   → déformation aléatoire du tracé

Stroke
  Color           → couleur des ondes
  Fade In / Fade Out → apparition et disparition progressives
  Start Width / End Width → épaisseur au début et à la fin de la durée de vie
```

**Usage typique — onde radar :**

```
Wave Type         = Polygon (4 côtés = carré, côtés arrondis)
Frequency         = 0.5 ondes/seconde
Expansion         = 150
Lifespan          = 3
Start Width       = 2
End Width         = 0
Fade Out          = 0.7
Color             = vert néon
```

**Usage typique — perturbation de surface :**

```
Frequency         = 3
Expansion         = 80
Lifespan          = 1.5
Start Width       = 8
End Width         = 0
Wobble Amount     = 20  ← les ondes ne sont pas des cercles parfaits
Blend Mode        = Screen, posé sur un fond sombre
```

# Combiner les effets génératifs

La puissance des effets génératifs dans AE ne vient pas d'un seul effet isolé, mais de leur **combinaison** — chaque effet alimentant ou modifiant le suivant.

**Exemple — fond ambiant organique :**

```
Calque solide noir
├── 4-Color Gradient            → définit la palette de base
├── Fractal Noise               → ajoute de la texture
│   (Blend Mode = Soft Light, Opacity = 40%)
└── Calque d'ajustement
    └── Fast Box Blur (rayon 20) → adoucit l'ensemble
```

**Exemple — particules qui révèlent un fond :**

```
Calque 1 : CC Particle World     → génère les particules (posé en Screen)
Calque 2 : Fractal Noise animé  → utilisé comme Luma Matte du calque suivant
Calque 3 : 4-Color Gradient     → fond révélé progressivement par le bruit
```

**Exemple — logo qui se trace avec une aura :**

```
Calque 1 : Solide + masque logo + Stroke (End animé 0→100%)
           → le tracé du logo
Calque 2 : Copie du calque 1 + Fast Box Blur (rayon 15) + Mode Screen
           → l'aura lumineuse floue autour du tracé
```

> 💡 La meilleure façon d'explorer les effets génératifs est d'expérimenter en appliquant plusieurs effets sur le même calque et en changeant leurs modes de fusion. Des combinaisons inattendues produisent régulièrement des résultats intéressants que l'on n'aurait pas imaginés a priori. C'est l'une des rares zones d'AE où "tester au hasard" est une méthode de travail légitime.
