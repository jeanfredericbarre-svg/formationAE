# La logique de la correction dans AE

La correction colorimétrique dans AE suit une hiérarchie : on corrige d'abord (Levels, Curves), puis on grade (couleurs, ambiance). Cette distinction est importante :

- **Corriger** → ramener l'image à un état neutre, fidèle à la réalité
- **Grader** → imposer une ambiance colorée, une intention artistique

> 💡 En pratique dans le motion design, on ne part pas toujours d'un footage à corriger — on grade directement des éléments graphiques pour créer une cohérence visuelle. La même logique s'applique : on part du neutre, on pousse dans une direction.

**L'ordre des effets dans la timeline AE est fondamental.** Les effets s'appliquent de haut en bas sur le calque. Mettre un Hue/Saturation avant un Curves ne donne pas le même résultat qu'après.

Ordre recommandé pour une correction complète :

```
1. Levels ou Curves        → correction des niveaux (noir, blanc, gamma)
2. Hue/Saturation          → ajustement des teintes et saturation
3. Color Balance           → couleurs dans les ombres/midtones/highlights
4. Tritone ou Gradient Map → colorisation stylisée (si besoin)
```

# Levels — la base de la correction

**Effect → Color Correction → Levels**

Levels contrôle les **niveaux d'entrée et de sortie** d'un calque — noir, blanc, et gamma (tons moyens). C'est l'effet de correction de base, rapide à utiliser.

**Interface :**

L'histogramme affiche la distribution des pixels. Les poignées sous l'histogramme contrôlent :

```
Input Black    → le niveau d'entrée que AE considère comme "noir pur"
                 déplacer vers la droite → assombrit les ombres, augmente le contraste
Input White    → le niveau d'entrée que AE considère comme "blanc pur"
                 déplacer vers la gauche → éclaircit les highlights, augmente le contraste
Gamma (1.0)    → la poignée centrale — contrôle les tons moyens
                 > 1.0 → image plus claire / < 1.0 → image plus sombre
Output Black   → le niveau de sortie minimum (évite les noirs purs)
Output White   → le niveau de sortie maximum (évite les blancs purs)
```

**Usage typique — récupérer une image trop terne :**

Une image qui ne touche pas les bords de l'histogramme manque de contraste. Ramener Input Black et Input White aux extrêmes de l'histogramme redonne du punch à l'image.

**Par canal :** le menu **Channel** permet d'appliquer Levels séparément sur R, G, B ou Alpha. Utile pour corriger une dominante colorée :

```
Channel = Red → déplacer Input White vers la gauche → plus de rouge dans les highlights
```

# Curves — la correction précise

**Effect → Color Correction → Curves**

Curves est la version avancée de Levels. Au lieu de trois points de contrôle fixes, on dessine **une courbe libre** qui mappe chaque valeur d'entrée vers une valeur de sortie.

**Lire la courbe :**

```
Axe X → valeur d'entrée (0 = noir, à droite = blanc)
Axe Y → valeur de sortie (0 = noir, en haut = blanc)
Courbe droite diagonale → aucune modification
```

**Les manipulations fondamentales :**

```
Tirer le milieu de la courbe vers le haut    → éclaircit les tons moyens
Tirer le milieu de la courbe vers le bas     → assombrit les tons moyens
Courbe en S                                  → augmente le contraste
  (ombre vers le bas + highlight vers le haut)
Point noir relevé (bas gauche vers le haut)  → aplatit les noirs, look cinématique
```

**Par canal :** comme Levels, Curves fonctionne par canal R, G, B indépendamment. C'est là que le grading colorimétrique précis se construit :

```
Canal Bleu  → courbe vers le bas dans les ombres → ombres orangées/chaudes
Canal Bleu  → courbe vers le haut dans les highlights → highlights froids/bleutés
→ Résultat : look "orange & teal" très courant dans le cinéma
```

> 💡 Curves est l'outil de grading le plus puissant et le plus précis dans AE. Prendre le temps de le maîtriser remplace avantageusement la plupart des autres effets de correction.

# Hue/Saturation — cibler une couleur

**Effect → Color Correction → Hue/Saturation**

Hue/Saturation modifie la teinte, la saturation et la luminosité — soit **globalement**, soit **en ciblant une couleur précise**.

**Les paramètres en mode global (Channel Control = Master) :**

```
Master Hue        → décale toutes les teintes (rotation du cercle chromatique)
Master Saturation → intensité des couleurs (-100 = noir et blanc, +100 = très saturé)
Master Lightness  → luminosité (différent de l'exposition — éviter les extrêmes)
Colorize          → force une teinte uniforme sur tout le calque (effet sépia, duotone)
```

**Le ciblage par canal — la vraie puissance :**

Le menu **Channel Control** permet de choisir une plage de couleur : Reds, Yellows, Greens, Cyans, Blues, Magentas.

```
Channel Control = Reds
Master Hue = +30    → seuls les rouges de l'image sont décalés vers l'orange
```

**Les barres de sélection en bas de l'effet** définissent précisément quelles teintes sont affectées et avec quelle transition. Elles peuvent être ajustées manuellement :

```
Barre intérieure    → zone pleinement affectée
Barre extérieure    → zone de transition (feather)
```

**Usage typique — remplacer une couleur précise :**

Changer la couleur d'un vêtement, d'un logo, d'une voiture dans un footage :

1. Channel Control → choisir la couleur dominante de l'objet cible
2. Ajuster les barres de sélection pour isoler précisément cette couleur
3. Modifier Master Hue pour décaler vers la couleur souhaitée
4. Ajuster Saturation si nécessaire

# Color Balance — les trois zones tonales

**Effect → Color Correction → Color Balance**

Color Balance permet de pousser des couleurs différentes dans les **ombres**, les **tons moyens** et les **highlights** indépendamment. C'est l'effet de grading le plus direct pour créer une ambiance.

**Les paramètres :**

```
Shadow Red/Green/Blue Balance    → couleur dans les ombres
Midtone Red/Green/Blue Balance   → couleur dans les tons moyens
Highlight Red/Green/Blue Balance → couleur dans les highlights
Preserve Luminosity              → maintient la luminosité globale pendant la correction
                                   → à laisser activé dans la majorité des cas
```

**Usage typique — look cinématique chaud :**

```
Shadow Blue Balance      = +20   → ombres légèrement froides
Midtone Red Balance      = +10   → tons moyens légèrement chauds
Highlight Red Balance    = +15   → highlights chauds (coucher de soleil)
Highlight Blue Balance   = -10   → retire le bleu des highlights
Preserve Luminosity      = ✅
```

> 💡 Color Balance (HLS) est une version alternative avec des contrôles différents. Dans la pratique, Color Balance standard est plus intuitif pour le grading.

# Tritone — la colorisation en trois tons

**Effect → Color Correction → Tritone**

Tritone remplace les couleurs d'un calque par **trois couleurs choisies**, mappées sur les ombres, les tons moyens et les highlights.

```
Highlights → couleur des zones les plus lumineuses
Midtones   → couleur des tons moyens
Shadows    → couleur des zones les plus sombres
Blend With Original → opacité du résultat (0% = tritone pur, 100% = original)
```

**Usage typique — créer une palette cohérente :**

Sur un calque de texture ou de bruit, Tritone permet de le coloriser exactement dans la palette du projet — sans chercher à correspondre visuellement.

**Usage typique — effet duotone :**

```
Highlights = blanc ou couleur claire
Midtones   = couleur intermédiaire (peut être identique aux highlights)
Shadows    = couleur sombre ou couleur accentuée
```

> 💡 Pour un effet duotone simple (deux couleurs seulement), il suffit de mettre Midtones à la même couleur que Highlights ou Shadows.

# Gradient Map — colorisation par luminance

**Effect → Color Correction → Gradient Map**

Gradient Map remplace les valeurs de luminance du calque par un **dégradé de couleurs**. Les pixels sombres prennent la couleur gauche du dégradé, les pixels clairs prennent la couleur droite.

```
Gradient Used For Colorizing → le dégradé (cliquable pour l'éditer)
Apply To Alpha               → applique aussi au canal alpha
```

**Différence avec Tritone :** Tritone a trois points fixes (ombres/midtones/highlights). Gradient Map a autant de points que souhaité, positionnables librement sur la plage de luminance.

**Usage typique — thermique, infrarouge :**

```
Dégradé : bleu foncé → violet → rouge → orange → jaune → blanc
→ simule un rendu caméra thermique
```

**Usage typique — stylisation graphique d'un footage :**

Un footage en niveaux de gris + Gradient Map avec les couleurs de la charte graphique = une stylisation immédiate et cohérente.

# Lumetri Color — le grading complet

**Effect → Color Correction → Lumetri Color**

Lumetri Color est le moteur de grading d'Adobe, partagé avec Premiere Pro. Il regroupe dans un seul panneau tous les outils de correction et de grading professionnels.

**Les sections de Lumetri Color :**

```
Basic Correction
  Tone            → Exposure, Contrast, Highlights, Shadows, Whites, Blacks
  Saturation      → saturation globale
  White Balance   → Temperature et Tint pour corriger la balance des blancs

Creative
  Faded Film      → aplatit les noirs pour un look "vieux film"
  Sharpen         → netteté globale
  Vibrance        → saturation qui protège les tons chair
  Split Toning    → couleurs dans les ombres et highlights séparément
  HSL Secondary   → correction isolée d'une plage de couleur précise

Curves
  RGB Curves      → équivalent de l'effet Curves, mais dans Lumetri
  Hue Saturation Curves → 5 types de courbes couleur avancées

Color Wheels & Match
  Lift (ombres) / Gamma (midtones) / Gain (highlights)
  → roues chromatiques pour pousser des couleurs dans chaque zone tonale

Vignette
  Amount    → intensité (négatif = vignette sombre)
  Midpoint  → rayon de la vignette
  Feather   → douceur des bords
```

**Pourquoi Lumetri plutôt que les effets séparés ?**

Un seul effet, un seul panneau, une interface cohérente. Sur un calque d'ajustement qui s'applique à toute une composition, Lumetri Color est le choix le plus efficace pour unifier l'ambiance.

> 💡 Lumetri Color n'est pas GPU-accéléré dans AE (contrairement à Premiere). Sur des compositions lourdes, si la performance est critique, les effets natifs individuels (Levels, Curves) restent plus légers.

# Calque d'ajustement — appliquer à toute une composition

Un **calque d'ajustement** (Adjustment Layer) est un calque transparent qui applique ses effets à **tous les calques situés en dessous** dans la timeline — sans toucher aux calques individuels.

**Créer un calque d'ajustement :**
`Layer → New → Adjustment Layer` ou `Ctrl/Cmd + Alt + Y`

**Usage type — ambiance globale en un seul calque :**

```
Timeline :
├── [AJ] Grading global     ← calque d'ajustement avec Lumetri Color
├── Texte
├── Logo
├── Background vidéo
└── Fond
```

Modifier le Lumetri Color sur le calque d'ajustement change l'ambiance de toute la composition — texte, logo, vidéo, fond — simultanément.

**Restreindre l'effet à une zone :** un masque sur le calque d'ajustement limite les effets à la zone masquée. Utile pour une vignette personnalisée ou une correction locale.

**Restreindre l'effet à certains calques :** repositionner le calque d'ajustement dans la timeline — les effets ne s'appliquent qu'aux calques **en dessous** de lui.

> 💡 La bonne pratique professionnelle : toujours terminer une composition avec au minimum un calque d'ajustement de grading en haut de la timeline. Même une légère correction (contraste +5, saturation +10) unifie visuellement une composition qui contient des éléments d'origines différentes.
