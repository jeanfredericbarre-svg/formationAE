## Les modes de fusion : logique générale 

Un mode de fusion (blending mode) définit **comment les pixels d'un calque se combinent avec les pixels des calques situés en dessous**. Sans mode de fusion (mode Normal), le calque supérieur recouvre simplement les calques inférieurs en fonction de son opacité.

> 💡 Analogie : imagine deux diapositives superposées sur un rétroprojecteur. En mode Normal, la diapositive du dessus cache celle du dessous. Les modes de fusion sont des règles mathématiques qui mélangent les couleurs des deux diapositives pixel par pixel.

**Où les trouver :** colonne **Mode** dans la timeline (`F4` pour afficher la colonne si elle n'est pas visible).

**La logique par groupes :**

| Groupe | Effet général | Modes principaux |
|---|---|---|
| Normal | Pas de fusion | Normal, Dissolve |
| Assombrissement | Assombrit le résultat | Multiply, Darken, Color Burn |
| Éclaircissement | Éclaircit le résultat | Screen, Add, Lighten |
| Contraste | Augmente le contraste | Overlay, Soft Light, Hard Light |
| Inversion | Effets d'inversion | Difference, Exclusion |
| Compositing | Couleur/luminance séparés | Hue, Saturation, Color, Luminosity |

## Les modes essentiels 

### Multiply — l'assombrissement pur

Multiply multiplie les valeurs de couleur. Le résultat est toujours **plus sombre ou égal** à l'original.

```
Blanc × n'importe quoi = n'importe quoi  (blanc est neutre en Multiply)
Noir × n'importe quoi  = noir            (noir reste noir)
```

**Usage concret :** poser une texture papier, bois, ou grain de film **au-dessus d'un fond coloré**. En Multiply, le fond de la texture (blanc ou clair) disparaît, et seules les zones sombres de la texture s'ajoutent au fond.

```
Timeline :
├── Texture papier  [Mode : Multiply, Opacity : 40%]
└── Fond coloré
→ Le fond coloré reçoit la texture sans que le blanc de la texture cache la couleur
```

### Screen — l'éclaircissement pur

Screen est l'inverse de Multiply. Le résultat est toujours **plus clair ou égal** à l'original.

```
Noir × n'importe quoi  = n'importe quoi  (noir est neutre en Screen)
Blanc × n'importe quoi = blanc           (blanc reste blanc)
```

**Usage concret :** ajouter des **éléments lumineux** (feu, fumée, particules, lens flares, étincelles) sur un fond. Ces éléments sont filmés sur fond noir — en Screen, le fond noir disparaît, et seule la lumière s'ajoute à la composition.

```
Timeline :
├── Particules d'étincelles [Mode : Screen]
├── Fumée                   [Mode : Screen]
└── Scène principale
→ Le fond noir des éléments disparaît, les lights s'ajoutent naturellement
```

### Add — la lumière additive

Add est similaire à Screen mais mathématiquement différent : il **additionne directement** les valeurs de pixels. Le résultat peut saturer (dépasser le blanc) ce qui crée des zones brûlées très lumineuses.

```
Screen = 1 - (1-A) × (1-B)   → résultat plafonné naturellement
Add    = A + B                → peut dépasser 1 (saturation)
```

**Usage concret :** éléments lumineux très intenses — lasers, néons, éclairs, halos. Là où Screen garde une certaine retenue, Add brûle et sature — ce qui renforce l'impression d'intensité lumineuse.

### Overlay — le contraste composé

Overlay combine Multiply et Screen selon la luminosité du calque du dessous :
- Zones sombres du dessous → comportement Multiply (assombrit)
- Zones claires du dessous → comportement Screen (éclaircit)

**Résultat :** augmentation du contraste, avec les couleurs du calque supérieur qui teintent légèrement le résultat.

**Usage concret :** ajouter une **texture de lumière ou d'ombre** sur une scène. Un dégradé ou une texture posée en Overlay renforce le contraste de l'image sans changer sa teinte globale.

### Soft Light — le contraste doux

Soft Light fonctionne comme Overlay mais avec un résultat plus doux, moins contrasté. C'est le mode pour des ajustements subtils d'ambiance lumineuse.

**Usage concret :** calque de correction colorimétrique peinte à la main (texture de lumière froide sur une demi-composition), posé en Soft Light pour une intégration douce.

### Difference — l'inversion relative

Difference soustrait les couleurs les plus foncées des plus claires. Là où les deux calques sont identiques, le résultat est noir. Là où ils diffèrent, on obtient des couleurs.

**Usage concret :** comparer deux versions d'un calque pour détecter les différences. En pratique dans le motion design : effets psychédéliques, transitions abstraites, effets de glitch.

### Hue / Saturation / Color / Luminosity

Ces quatre modes permettent de fusionner **une seule composante** de la couleur :

```
Hue         → la teinte du calque supérieur, la saturation et luminosité du dessous
Saturation  → la saturation du calque supérieur, teinte et luminosité du dessous
Color       → teinte et saturation du calque supérieur, luminosité du dessous
Luminosity  → la luminosité du calque supérieur, teinte et saturation du dessous
```

**Usage concret — coloriser en Color :**

Pour coloriser un footage en niveaux de gris : créer un solide de couleur au-dessus en mode Color. La couleur du solide s'applique en conservant la luminance du footage.

```
Timeline :
├── Solide orange   [Mode : Color, Opacity : 80%]
└── Footage noir et blanc
→ Footage colorisé en orange, avec toutes ses nuances de gris préservées
```

## Modes de fusion et expressions 

Le mode de fusion d'un calque n'est pas animable directement — mais on peut simuler une transition entre deux modes en créant deux versions du calque et en animant leur opacité.

Une approche plus élégante consiste à utiliser les propriétés **Blend Mode** et **Opacity** ensemble :

```javascript
// Transition d'opacité qui simule l'apparition progressive d'un Screen :
// Sur l'Opacity du calque en mode Screen :
ease(time, 1, 3, 0, 100)
// Le calque Screen "apparaît" de 0 à 100% entre les secondes 1 et 3
```

## Track Matte — utiliser un calque comme masque 

Le Track Matte est un système qui utilise la **luminance ou l'alpha** d'un calque pour **définir la transparence d'un autre calque**.

> 💡 Analogie : le Track Matte est un pochoir. Le calque matte est le pochoir ; le calque en dessous est la peinture. La forme du pochoir détermine où la peinture apparaît.

**Mise en place :**

```
Timeline (ordre obligatoire) :
├── Calque Matte    ← le pochoir (souvent rendu invisible automatiquement)
└── Calque cible    ← la peinture — TrkMat → Alpha Matte ou Luma Matte
```

Le calque matte doit être **directement au-dessus** du calque cible. Dans la colonne **TrkMat** (Track Matte) du calque cible :

```
No Track Matte      → pas de matte
Alpha Matte         → les zones opaques du calque matte = zones visibles de la cible
Alpha Inverted Matte → inverse : zones transparentes du matte = zones visibles
Luma Matte          → les zones blanches du matte = zones visibles
Luma Inverted Matte → les zones sombres = zones visibles
```

**Alpha Matte vs Luma Matte :**

- **Alpha Matte** → utilise le canal alpha du calque matte. Parfait pour des textes ou des shapes qui ont déjà un contour propre.
- **Luma Matte** → utilise la luminance. Parfait pour des footages (nuages, fumée, explosion) qui n'ont pas de canal alpha mais dont les zones claires définissent la forme.

<span id="combinaisons"></span>
## Combinaisons avancées 

### Texte rempli par une vidéo

```
Timeline :
├── Texte           ← calque matte (le texte est le pochoir)
└── Vidéo           ← TrkMat = Alpha Matte
                       → la vidéo n'apparaît que dans la forme des lettres
```

### Révélation par une fumée (Luma Matte)

```
Timeline :
├── Fumée / nuage   ← calque matte (filmé sur fond noir)
└── Image révélée   ← TrkMat = Luma Matte
                       → l'image apparaît là où la fumée est blanche/claire
                       → bords organiques et naturels, aucun masque à dessiner
```

### Shape Layer animé comme matte

```
Timeline :
├── Cercle animé (Scale 0→100%) ← calque matte
└── Composition entière         ← TrkMat = Alpha Matte
→ La composition entière apparaît à travers le cercle qui grandit
→ Transition "iris" sans avoir à masquer chaque calque individuellement
```

### Track Matte + mode de fusion

Combiner les deux systèmes multiplie les possibilités. Un calque peut simultanément avoir un Track Matte **et** un mode de fusion.

```
Timeline :
├── Bruit / texture  ← calque matte
└── Solide coloré    ← TrkMat = Luma Matte + Mode = Screen
→ La texture découpe le solide, et le solide s'ajoute en Screen à la comp
→ Effet de lumière organique texturée
```

> 💡 Quand un Track Matte est activé sur un calque, le calque matte est automatiquement rendu invisible dans la composition. Si tu as besoin que le calque matte soit aussi visible, duplique-le et garde l'original visible au-dessus de la paire matte/cible.
