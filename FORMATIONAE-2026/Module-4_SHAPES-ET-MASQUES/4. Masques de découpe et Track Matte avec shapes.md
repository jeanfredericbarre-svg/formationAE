<span id="masque-vs-shape"></span>
# Masque et Shape Layer : deux choses différentes

Avant d'aller plus loin, il faut lever une confusion fréquente. Dans AE, le mot "masque" peut désigner deux choses très différentes :

**Un masque** (`Mask`) est un tracé dessiné **directement sur un calque existant**. Il découpe ou modifie la zone visible de ce calque. Il appartient au calque — il n'est pas un calque indépendant.

**Un Shape Layer** est un **calque à part entière**, avec ses propres propriétés de transformation, ses propres effets, sa propre place dans la timeline. Ses tracés peuvent servir de masque, mais aussi d'éléments visuels autonomes.

```
Calque vidéo
└── Mask 1  ← tracé de masque : découpe le calque depuis l'intérieur
            ← pas visible dans la timeline comme calque séparé

Shape Layer  ← calque indépendant
└── Contenu
    └── Ellipse 1  ← forme vectorielle visible, animable, réutilisable
                   ← peut aussi servir de Track Matte pour un autre calque
```

<span id="masque-decoupe"></span>
# Le masque de découpe sur un Shape Layer

Un masque peut être appliqué à n'importe quel calque — y compris un Shape Layer. C'est utile pour découper une forme vectorielle avec un contour précis, tout en conservant les avantages du Shape Layer (Trim Paths, Repeater, expressions...).

**Comment ajouter un masque à un Shape Layer :**

1. Sélectionner le Shape Layer dans la timeline
2. Utiliser l'outil Rectangle ou l'outil Plume **dans la barre d'outils** — et s'assurer qu'aucun groupe n'est sélectionné dans le calque, sinon AE crée une nouvelle forme dans le groupe plutôt qu'un masque

```
// Règle critique : outil de forme + calque sélectionné mais aucun groupe actif = masque
//                  outil de forme + groupe actif dans le Shape Layer = nouvelle forme dans le groupe
```

Le masque apparaît alors sous la propriété **Masques** du calque, séparément du **Contenu**.

```
Shape Layer
├── Masques
│   └── Masque 1   ← le tracé de découpe
└── Contenu
    └── Groupe 1   ← le contenu vectoriel découpé
```

**Cas d'usage typique :** un Repeater qui génère une grille de formes, découpée par un masque circulaire pour ne révéler que la zone centrale.

<span id="modes-masque"></span>
# Les modes de masque

Chaque masque possède un **mode** qui détermine comment il interagit avec le calque et avec les autres masques. Le mode se règle dans la colonne **Mode** de la timeline, à gauche du nom du masque.

```
Aucun       → le masque est désactivé (tracé visible mais sans effet)
Ajouter     → zone DANS le masque = visible  (comportement par défaut)
Soustraire  → zone DANS le masque = invisible (découpe un trou)
Intersect   → seule la zone commune entre ce masque et les précédents = visible
Éclaircir   → fusionne en gardant les zones les plus claires (rare)
Assombrir   → fusionne en gardant les zones les plus sombres (rare)
Différence  → inverse l'intersection (zones non communes = visibles)
```

**Combiner plusieurs masques :** la puissance vient de la combinaison. Un masque en mode `Ajouter` définit la zone visible de base, un second en mode `Soustraire` y perce un trou.

```
Masque 1 : Rectangle large — mode Ajouter     → zone visible : le rectangle
Masque 2 : Ellipse centrale — mode Soustraire → perce un trou circulaire au centre
→ résultat : un cadre rectangulaire avec une fenêtre ovale évidée
```

Chaque masque possède aussi une propriété **Contour du masque** (Mask Feather) pour adoucir ses bords, et **Opacité du masque** pour le rendre partiellement transparent.

<span id="animer-masque"></span>
# Animer un masque

Les masques sont entièrement animables — leur forme, leur position, leur expansion, leur opacité. C'est ce qui permet les effets de révélation ou de cache animé.

## Animer la forme du masque

Chaque point du tracé d'un masque peut être keyframé individuellement. Dans la timeline, déplier **Masques → Masque 1 → Tracé du masque** et activer le chronomètre.

**Effet de révélation progressif (wipe) :**

```
t = 0s : Masque Rectangle → largeur = 0px  (masque vide, rien de visible)
t = 1s : Masque Rectangle → largeur = 1920px (masque plein, tout visible)
→ le contenu du Shape Layer se révèle de gauche à droite
```

## Expansion du masque

La propriété **Expansion du masque** (Mask Expansion) agrandit ou réduit le masque uniformément vers l'intérieur ou l'extérieur, sans modifier le tracé d'origine.

```
Expansion : 0px   → masque à sa taille normale
Expansion : 20px  → masque agrandi de 20px sur tous les côtés
Expansion : -20px → masque rétréci de 20px sur tous les côtés
```

**Usage typique :** combiner avec le Contour du masque pour un effet de halo ou de vignette animée.

> 💡 Animer l'Expansion est bien plus souple qu'animer le tracé point par point quand on veut simplement "ouvrir" ou "fermer" un masque de manière uniforme.

<span id="trackmatte-principe"></span>
# Le Track Matte : principe

Le **Track Matte** (Matte de Suivi) est un système qui utilise **un calque pour contrôler la transparence d'un autre calque**. Contrairement au masque qui appartient au calque, le Track Matte est un calque indépendant dans la timeline.

**La règle de fonctionnement :**

```
Calque du dessus  → le calque Matte (il définit la forme de la découpe)
Calque du dessous → le calque affecté (il est découpé par le Matte)
```

Le calque Matte est **automatiquement désactivé visuellement** quand il est assigné comme Track Matte — il disparaît du rendu mais continue de définir la découpe.

**Comment l'assigner :** dans la timeline, sur le calque du dessous, trouver la colonne **TrkMat** (Track Matte). Si elle n'est pas visible : clic droit sur l'en-tête → **Colonnes** → **Modes**. Choisir le mode dans le menu déroulant.

```
Timeline :
  Calque A (Shape Layer rectangle)   ← le Matte
  Calque B (vidéo ou image)          ← TrkMat : "Alpha Matte Calque A"
→ La vidéo B n'est visible qu'à l'intérieur du rectangle de A
```

<span id="alpha-vs-luma"></span>
# Track Matte Alpha vs Luma

AE propose quatre modes de Track Matte, qui diffèrent par ce qu'ils utilisent comme source de transparence.

```
Alpha Matte          → utilise le canal alpha du calque Matte
                        zones opaques du Matte = visibles dans le calque affecté
                        zones transparentes du Matte = invisibles

Alpha Matte inversé  → inverse : zones transparentes du Matte = visibles

Luma Matte           → utilise la luminosité du calque Matte
                        zones blanches (lumineuses) = visibles
                        zones noires = invisibles

Luma Matte inversé   → inverse : zones noires = visibles
```

**Quand utiliser Alpha :** pour des Shape Layers ou des calques avec transparence (PNG, formes vectorielles). Le canal alpha est net — la découpe est franche.

**Quand utiliser Luma :** pour des calques sans transparence (vidéo, solides, dégradés). La luminosité fait office de canal de transparence. Un dégradé blanc vers noir crée une transition douce entre visible et invisible.

```
Luma Matte avec un dégradé :
  Blanc = 100% visible
  Gris 50% = 50% visible (semi-transparent)
  Noir = 0% visible (invisible)
→ transition progressive et douce, impossible à obtenir avec Alpha Matte
```

<span id="shape-trackmatte"></span>
# Shape Layer comme Track Matte

Les Shape Layers sont des Mattes particulièrement puissants parce qu'ils sont vectoriels, animables et expressables. C'est la combinaison la plus courante en motion design.

**Mise en place :**

```
Timeline :
  Shape Layer (ellipse)   ← le Matte — sa forme définit la fenêtre visible
  Calque vidéo            ← TrkMat : "Alpha Matte Shape Layer"
→ La vidéo n'est visible qu'à l'intérieur de l'ellipse
```

**Avantages par rapport à un masque classique :**

Le masque est lié au calque — si on veut la même découpe sur 3 calques différents, il faut dupliquer et maintenir 3 masques séparés. Le Shape Layer Matte est indépendant : un seul Shape Layer peut servir de Matte pour plusieurs calques si on le duplique et qu'on l'assigne à chacun. Et surtout, il bénéficie de toutes les fonctionnalités des Shape Layers : Trim Paths, Repeater, opérations booléennes, expressions.

**Exemple : révélation circulaire avec Scale :**

```
Shape Layer : Ellipse, Scale animée de [0%, 0%] à [200%, 200%] sur 1 seconde
Calque vidéo : TrkMat Alpha Matte
→ la vidéo apparaît en s'ouvrant depuis le centre, comme un iris
```

<span id="trackmatte-anime"></span>
# Track Matte animé avec un Shape Layer

Tout ce qui est animable sur un Shape Layer l'est également quand il sert de Track Matte. C'est là que la combinaison devient réellement intéressante.

## Révélation par Trim Paths

Trim Paths sur le Matte révèle le calque sous-jacent en dessinant progressivement le contour de la forme.

```
Shape Layer Matte : cercle avec Trim Paths
  Fin : 0% à 0s → 100% à 1.5s
Calque photo : TrkMat Alpha Matte
→ la photo se révèle progressivement en suivant le contour du cercle
```

## Wipe avec un rectangle animé

```
Shape Layer Matte : rectangle
  Contenu → Rectangle 1 → Taille → W animée de 0 à 1920px sur 0.8s
Calque titre : TrkMat Alpha Matte
→ le titre se révèle de gauche à droite derrière un volet
```

## Matte avec Repeater pour des révélations en grille

```
Shape Layer Matte :
  Ellipse (petite, ex : 40×40px)
  Repeater : 10 copies × 10 rangées, espacement [192px, 108px]
  Scale du Repeater animée de [0%, 0%] à [100%, 100%]
Calque vidéo : TrkMat Alpha Matte
→ la vidéo apparaît en "pixels" circulaires qui grandissent simultanément
```

> 💡 Le Repeater dans un Matte est l'une des techniques les plus efficaces pour créer des transitions visuellement complexes avec un minimum de travail. Une seule forme, un seul Repeater, et la transition semble avoir nécessité des heures de travail manuel.

## Précomposer le Matte pour plus de flexibilité

Quand le Shape Layer Matte devient complexe (plusieurs groupes, expressions, effets), il peut être utile de le précomposer. La précomp devient alors le Matte.

```
Timeline :
  Précomp_Matte  ← contient le Shape Layer animé complexe
  Calque vidéo   ← TrkMat Alpha Matte Précomp_Matte
```

Cela permet aussi d'appliquer des effets sur le Matte lui-même — un flou gaussien sur la précomp Matte crée des bords doux sur la découpe, sans toucher au calque affecté.

<span id="choisir"></span>
# Masque ou Track Matte : quand choisir lequel

Les deux systèmes font des choses similaires — découper un calque — mais leurs forces sont différentes.

**Choisir un masque quand :**

```
→ la découpe est simple et propre à ce calque (pas besoin de la réutiliser)
→ la forme de découpe est liée au contenu du calque (ex : recadrer une vidéo)
→ on veut combiner plusieurs masques avec des modes différents (Ajouter + Soustraire)
→ on veut un contour adouci (Feather) sans calque supplémentaire
```

**Choisir un Track Matte quand :**

```
→ la découpe doit être animée de manière complexe (Trim Paths, Repeater, expressions)
→ la même forme de découpe doit servir pour plusieurs calques
→ le Matte doit bénéficier d'effets indépendants (flou, distorsion...)
→ on veut une révélation basée sur une vidéo ou un dégradé (Luma Matte)
→ la forme de découpe doit être modifiable indépendamment du contenu
```

**La règle empirique :**

> 💡 Si la découpe est statique ou presque, un masque suffit. Dès que la découpe est animée, vectorielle et potentiellement réutilisable, un Shape Layer en Track Matte est presque toujours la meilleure option.
