## C'est quoi le Repeater ? 

Le Repeater est un opérateur de groupe qui **duplique toutes les formes du groupe** un nombre paramétrable de fois, en appliquant un décalage cumulatif à chaque copie, décalage de position, rotation, échelle, opacité.

> 💡 Analogie : au lieu de créer 12 rectangles à la main pour faire une horloge, tu crées un seul rectangle, tu ajoutes un Repeater avec 12 copies, et tu règles un décalage de rotation de 30°, AE place les 12 rectangles automatiquement.

Le Repeater est **non-destructif** : tu modifies le groupe original, toutes les copies se mettent à jour. Tu modifies les paramètres du Repeater, le motif entier se recalcule.

**Pour l'ajouter :** dans la timeline d'un groupe → bouton **Ajouter** → **Répéteur**.

<span id="parametres"></span>
## Les paramètres du Repeater 

```
Copies          → nombre total de copies (inclut l'original)
Décalage        → décale l'index de la première copie
                  (utile pour animer les copies en séquence)

Transform — Répéteur :
  Point d'ancrage → pivot autour duquel le décalage s'applique
  Position        → décalage de position cumulatif entre chaque copie
  Rotation        → décalage de rotation cumulatif
  Échelle         → décalage d'échelle cumulatif (en %)
  Opacité début   → opacité de la première copie
  Opacité fin     → opacité de la dernière copie
```

**"Cumulatif"** est le mot clé. Si le décalage de Position est `[50, 0]` et qu'il y a 5 copies :
- Copie 1 : position d'origine
- Copie 2 : +50px
- Copie 3 : +100px
- Copie 4 : +150px
- Copie 5 : +200px

<span id="patterns"></span>
## Patterns courants 

### Ligne de formes

```
Copies   = 8
Position = [60, 0]   ← espacement horizontal de 60px
Rotation = 0°
```
→ 8 copies alignées horizontalement, espacées de 60px.

### Cercle de points (horloge, loader)

```
Point d'ancrage = [0, -100]   ← décale le pivot 100px au-dessus du centre
Copies          = 12
Position        = [0, 0]
Rotation        = 30°          ← 360° / 12 copies
```
→ 12 copies disposées en cercle de rayon 100px, comme les chiffres d'une horloge.

> 💡 Le Point d'ancrage du Repeater est la clé pour les dispositions circulaires. En décalant le pivot loin du centre de la forme, chaque rotation devient une révolution autour d'un axe éloigné, ce qui place les copies sur un cercle.

### Spirale

```
Copies   = 20
Position = [10, 0]
Rotation = 18°         ← 360° / 20 = 18° par copie
Échelle  = [95%, 95%]  ← chaque copie est légèrement plus petite
```
→ 20 copies qui s'enroulent en spirale en rétrécissant progressivement.

### Fondu de bord

```
Opacité début = 100%
Opacité fin   = 0%
```
→ les copies s'effacent progressivement de la première à la dernière.

<span id="expressions"></span>
## Repeater et expressions 

**Nombre de copies piloté par un Slider :**

```javascript
// Sur la propriété "Copies" :
Math.round(thisComp.layer("CONTROLS").effect("Nb copies")("Curseur"))
// Math.round() car Copies n'accepte que des entiers
```

**Décalage animé : les copies arrivent en séquence :**

La propriété **Décalage** du Repeater, combinée à des keyframes ou une expression, décale l'animation de chaque copie dans le temps :

```javascript
// Sur la propriété "Décalage" :
time * -5
// Chaque frame, le décalage augmente — les copies semblent "défiler"
// Le signe négatif inverse le sens du défilement
```

**Rotation continue — loader circulaire :**

```javascript
// Sur la propriété Rotation du groupe (pas du Repeater) :
time * 120
// Le groupe entier tourne à 120°/seconde
// Combiné à un Repeater en cercle avec Opacité début/fin différents
// → loader "spinner" classique
```

> 💡 La combinaison Repeater + Trim Paths est particulièrement puissante : Trim Paths contrôle la longueur visible de chaque copie, le Repeater les distribue en cercle — résultat : un loader animé en quelques minutes, entièrement procédural.
