## Pourquoi gérer son footage ? <span id="pourquoi"></span>

After Effects est un logiciel de composition, pas de montage. Il n'est pas optimisé pour lire des fichiers vidéo compressés (H.264, HEVC…) à la volée. Une mauvaise gestion du footage est la première cause de lenteur sur un projet.

---

## Les proxies <span id="proxies"></span>

Un **proxy** est un fichier de substitution, plus léger, que AE utilise pendant la composition. Au rendu final, AE rebascule automatiquement sur le fichier original haute qualité.

### Créer un proxy

1. Dans le panneau **Projet**, clic droit sur le footage → **Create Proxy** → **Movie** ou **Still**
2. Choisir un codec léger : **ProRes 422 LT** ou **DNxHD** (éviter H.264)
3. Une résolution réduite (½ ou ¼) est souvent suffisante

> Le proxy est indiqué par un carré vide ☐ à gauche du fichier dans le panneau Projet. Quand il est actif : ☑.

### Activer / désactiver les proxies

- Clic sur le carré ☐/☑ dans le panneau Projet pour basculer fichier par fichier
- Bouton **Toggle Proxies** dans la barre du panneau pour tout basculer d'un coup

## Résolution d'aperçu <span id="apercu"></span>

Indépendamment des proxies, la **résolution d'aperçu** dans le panneau Composition contrôle la qualité de l'affichage en temps réel :

| Réglage | Usage |
|---|---|
| **Full** | Rendu final uniquement |
| **Half** | Composition courante, bon compromis |
| **Quarter** | Projets lourds, vérification des timings |
| **Custom** | Cas spécifiques |

Raccourci : menu déroulant en bas du panneau Comp, ou `Maj + Ctrl/Cmd + J` pour les préférences de composition.

## Footage mal interprété <span id="interpretation"></span>

AE peut mal interpréter certains fichiers :
- **Alpha** : Prémultiplié vs. droit → clic droit sur le footage → **Interpret Footage** → **Main**
- **Frame rate** : si le footage est à 23.976 et le projet à 25fps, AE n'interpole pas automatiquement
- **Pixel aspect ratio** : problème fréquent avec du vieux footage DV (0.9091)

### Règle d'or

> Les réglages de **Interpret Footage** s'appliquent à la source, pas à la composition. Un même fichier réinterprété sera mis à jour partout où il est utilisé.

## Collect Files — archiver un projet <span id="archiver"></span>

Pour déplacer ou archiver un projet proprement :

**File → Dependencies → Collect Files**

- Regroupe le `.aep` + tous les sources dans un seul dossier
- Évite les liens cassés lors du transfert sur un autre poste ou vers un client

## Conseils pratiques <span id="conseils"></span>

- **Ne jamais travailler avec du H.264 natif** dans AE — transcoder en ProRes ou DNxHD avant de l'importer
- **Organiser le panneau Projet** avec des dossiers (solides, footage, précomps, audio) — un projet rangé se charge plus vite
- **Purger le cache RAM** régulièrement si AE ralentit en cours de session : `Ctrl/Cmd + Alt + /`
- **Snapshot** : `Maj + F5` capture une image de la comp, `F5` la réaffiche pour comparer avant/après un réglage
- **Auto-save** : Préférences → Auto-Save → toutes les 5 minutes, garder 10 versions
