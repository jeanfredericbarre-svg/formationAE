## Essential Graphics Panel 

Le panneau **Essential Graphics** (Propriétés essentielles) est l'interface qui permet de construire des **templates de motion design paramétrables**, des compositions AE dont certains paramètres sont modifiables sans ouvrir AE.

**Ouvrir le panneau :** Fenêtre → Propriétés essentielles.

Le panneau affiche deux onglets :
- **Modifier** : pour construire le template (ajouter des propriétés, les nommer, définir des plages)
- **Principaux** : pour voir les propriétés exposées d'une précomp sélectionnée dans la timeline

## Créer un template avec Master Properties 

**Étape 1 : Prépare ta composition**

Construis ton animation normalement. Identifie quels paramètres tu veux rendre modifiables : textes, couleurs, durées, images de fond...

**Étape 2 : Ouvre le panneau Essential Graphics**

Assure-toi que la composition à transformer en template est la composition active.

**Étape 3 : Ajoute des propriétés**

Dans la timeline, **glisse les propriétés directement dans le panneau Essential Graphics**, ou utilise le bouton **Ajouter une propriété**. Types supportés :

```
✅ Propriétés de calque texte (sourceText → devient un champ de texte éditable)
✅ Couleurs (Color Controls, couleur de fond...)
✅ Sliders (Slider Controls → valeur numérique éditable)
✅ Cases à cocher
✅ Images (footage → remplaçable par le monteur)
```

**Étape 4 : Nommer et organiser**

Dans le panneau, chaque propriété peut être renommée. Les noms apparaissent tels quels dans Premiere Pro.

```
"Titre principal"      ← sourceText d'un calque texte
"Sous-titre"           ← sourceText d'un autre calque texte
"Couleur fond"         ← Color Control
"Durée d'affichage"    ← Slider Control (en secondes)
```

Des **séparateurs** permettent de grouper les propriétés visuellement dans le panneau Premiere.

**Étape 5 : Définir les plages pour les Sliders**

Pour chaque Slider, définis une valeur minimale et maximale cohérentes avec l'usage, évite de laisser la plage par défaut (0–100) si la propriété pilotée ne correspond pas.

## Exporter en MOGRT 

**MOGRT** = Motion Graphics Template. C'est le format de fichier qui encapsule la composition AE et ses propriétés essentielles dans un fichier utilisable directement dans Premiere Pro ou After Effects.

**Exporter :** dans le panneau Essential Graphics → bouton **Exporter en tant que Motion Graphics Template**.

Options à l'export :
- **Destination** : disque local ou bibliothèque Creative Cloud partagée (pour partager avec une équipe)
- **Vignette** : aperçu miniature affiché dans Premiere

> 💡 Le fichier `.mogrt` est autonome, il contient tout ce dont Premiere a besoin. Le monteur n'a pas besoin d'avoir les fichiers sources du projet AE.

**Attention aux polices :** les polices utilisées dans la composition doivent être **installées sur la machine du monteur** pour s'afficher correctement. Les polices Adobe Fonts (anciennement Typekit) sont synchronisables via Creative Cloud.

## Utiliser un MOGRT dans Premiere Pro 

Dans Premiere Pro, le panneau **Graphiques essentiels** (Fenêtre → Graphiques essentiels) permet d'installer et d'utiliser les MOGRT.

**Workflow du monteur :**

1. Importer le `.mogrt` dans le panneau Graphiques essentiels
2. Faire glisser le template sur la timeline Premiere (comme un clip)
3. Sélectionner le clip → les propriétés exposées apparaissent dans le panneau
4. Modifier les textes, couleurs, images directement dans Premiere
5. Le rendu reflète les modifications, sans ouvrir AE

> 💡 C'est le workflow professionnel standard entre un motion designer et un monteur vidéo. Le motion designer crée l'animation une fois dans AE ; le monteur l'adapte à chaque projet depuis Premiere, sans risquer de "casser" l'animation sous-jacente.
>  💡 Attention au niveau de liberté laissé au monteur/journaliste : si vous laissez trop de liberté, la cohérence visuelle entre les monteurs risque d'être très aléatoire.
