## C'est quoi un Array ? <span id="array"></span>

Un **Array** (tableau) est une liste ordonnée de valeurs, regroupées dans un seul objet.

En JavaScript, un Array s'écrit entre crochets, avec les valeurs séparées par des virgules :

```javascript
[100, 200]       // un tableau de deux nombres
[0.5, 1, 0.8]    // un tableau de trois nombres
```

Chaque valeur a un **index** — un numéro de position qui commence à **0** :

```javascript
var position = [100, 200];

position[0]   // → 100  (le premier élément, X)
position[1]   // → 200  (le deuxième élément, Y)
```

> 💡 L'index commence toujours à **0**, pas à 1. C'est une convention universelle en programmation, pas spécifique à AE.

## Les propriétés multidimensionnelles dans AE <span id="multidim"></span>

Dans After Effects, de nombreuses propriétés stockent plusieurs valeurs à la fois. Ces propriétés sont des Arrays.

| Propriété | Type | Exemple |
|---|---|---|
| Position (2D) | `[x, y]` | `[960, 540]` |
| Position (3D) | `[x, y, z]` | `[960, 540, 0]` |
| Échelle | `[x, y]` | `[100, 100]` |
| Point d'ancrage | `[x, y]` | `[0, 0]` |
| Couleur | `[r, g, b, a]` | `[1, 0.5, 0, 1]` |

À l'inverse, certaines propriétés sont des **valeurs simples** (scalaires) — elles ne sont pas des Arrays :

| Propriété | Type | Exemple |
|---|---|---|
| Opacité | nombre | `100` |
| Rotation | nombre | `45` |
| Flou (intensité) | nombre | `20` |

> 💡 Comment savoir si une propriété est un Array ? Dans AE, si la propriété affiche plusieurs valeurs côte à côte dans la timeline (un X et un Y par exemple), c'est un Array.

## Lire une valeur — value[0], value[1] <span id="lire"></span>

Le mot-clé `value` dans une expression désigne **la valeur actuelle de la propriété**. Si cette propriété est un Array, `value` est lui-même un Array, et on peut en extraire un élément avec son index.

**Exemple : récupérer uniquement le X d'une Position**

Sur la propriété **Rotation** d'un calque, on veut que la rotation suive le X de la position d'un autre calque :

```javascript
var source = thisComp.layer("BALLE");
source.transform.position[0]   // récupère uniquement le X de la position de "BALLE"
```

**Exemple : déclencher quelque chose selon la position verticale**

```javascript
var posY = transform.position[1];   // le Y de ce calque

linear(posY, 0, 1080, 0, 100)       // mappe la position Y (0→1080) vers une opacité (0→100)
```

> 💡 `position[0]` → X (horizontal), `position[1]` → Y (vertical), `position[2]` → Z (profondeur, en 3D). Cette convention X/Y/Z correspond à l'ordre dans lequel les valeurs apparaissent dans le panneau de la timeline.

## Modifier une seule dimension <span id="modifier"></span>

C'est un besoin très courant : appliquer un `wiggle` uniquement sur l'axe vertical, sans toucher au X. Ou lier le Y à une expression, tout en laissant le X libre (animable avec des keyframes).

**Le problème :** une expression sur Position doit retourner un Array complet `[x, y]`. On ne peut pas retourner seulement `y`.

**La solution :** construire un nouvel Array en combinant `value[0]` (le X qu'on laisse tel quel) avec le calcul qu'on veut appliquer au Y.

```javascript
// Wiggle uniquement sur l'axe vertical (Y)
var agitation = wiggle(3, 80);          // wiggle sur les deux axes
[value[0], agitation[1]]                // on garde le X original, on prend le Y du wiggle
```

```javascript
// Oscillation verticale avec sin(), sans toucher au X
var amplitude = 50;
var frequence = 2;
var oscY = Math.sin(time * frequence * Math.PI * 2) * amplitude;

[value[0], value[1] + oscY]             // on ajoute l'oscillation uniquement sur le Y
```

> 💡 `value[0]` dans ce contexte désigne la valeur X **telle qu'elle est** — avec ses keyframes si il y en a. On ne la fige pas, on la laisse vivre normalement. On ne remplace que le Y.

## Reconstruire un tableau complet <span id="reconstruire"></span>

Parfois, on ne veut pas partir de `value` mais construire un Array entièrement depuis des calculs ou des sources externes.

**Exemple : Position pilotée par deux sliders séparés**

```javascript
var ctrlX = effect("Position X")("Curseur");
var ctrlY = effect("Position Y")("Curseur");

[ctrlX, ctrlY]    // on retourne un Array complet construit de zéro
```

**Exemple : Position qui suit un autre calque uniquement en X, et un slider pour le Y**

```javascript
var sourceX = thisComp.layer("GUIDE").transform.position[0];
var manualY  = effect("Hauteur")("Curseur");

[sourceX, manualY]
```

> 💡 Dès qu'une expression est activée sur une propriété, elle **prend le contrôle total** de cette propriété — les keyframes existants sont ignorés. Si tu veux combiner keyframes et expression, il faut passer par `value` comme vu dans la section précédente.

## Arrays et couleurs — le cas RGBA <span id="rgba"></span>

Les couleurs dans After Effects sont des Arrays de **4 valeurs** : Rouge, Vert, Bleu, Alpha. Mais contrairement à ce qu'on voit dans Photoshop ou CSS, ces valeurs ne vont **pas de 0 à 255** — elles vont **de 0 à 1**.

```javascript
[1, 0, 0, 1]       // rouge pur, opaque
[0, 0, 1, 1]       // bleu pur, opaque
[1, 1, 1, 0.5]     // blanc à 50% de transparence
[0, 0, 0, 1]       // noir opaque
```

**Exemple : lier la couleur d'un calque à celle d'un autre**

Sur la propriété **Couleur** d'un effet Remplissage, on peut écrire :

```javascript
thisComp.layer("PALETTE").effect("Remplissage")("Couleur")
// retourne directement l'Array [r, g, b, a] de l'autre calque
```

**Exemple : modifier uniquement l'alpha d'une couleur existante**

```javascript
var couleurSource = thisComp.layer("PALETTE").effect("Remplissage")("Couleur");

[couleurSource[0], couleurSource[1], couleurSource[2], 0.5]
// on conserve les canaux R, G, B et on force l'alpha à 0.5
```

> 💡 Pour convertir une valeur 0–255 en 0–1, divise par 255. Par exemple, `128 / 255` donne environ `0.5`. AE affiche parfois les valeurs en 0–255 dans l'interface, mais les expressions travaillent toujours en 0–1.

## Cas pratiques <span id="pratiques"></span>

**Faire osciller un calque uniquement à la verticale**

```javascript
// Sur la propriété Position
var amplitude = 40;
var vitesse   = 1.5;

[value[0], value[1] + Math.sin(time * vitesse * Math.PI * 2) * amplitude]
```

**Suivre le X d'un autre calque, garder son propre Y**

```javascript
// Sur la propriété Position
var cible = thisComp.layer("LEADER");

[cible.transform.position[0], value[1]]
```

**Créer un tremblement de caméra vertical uniquement**

```javascript
// Sur la propriété Position d'une caméra ou d'un Null
var secousse = wiggle(8, 30);

[value[0], secousse[1]]
```

**Rendre une couleur plus transparente au fil du temps**

```javascript
// Sur la propriété Couleur d'un effet Remplissage
var base = [1, 0.4, 0.1, 1];    // une couleur orange opaque
var alpha = linear(time, 0, 3, 1, 0);   // de 1 à 0 entre 0s et 3s

[base[0], base[1], base[2], alpha]
```
