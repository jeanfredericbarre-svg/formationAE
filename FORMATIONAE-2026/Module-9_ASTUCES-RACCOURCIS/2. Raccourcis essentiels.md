# Raccourcis essentiels

Les raccourcis ci-dessous sont ceux qu'un utilisateur d'After Effects utilise quotidiennement. Les maîtriser divise le temps de travail par deux.

---

## Navigation dans la timeline

| Raccourci | Action |
|---|---|
| `Espace` | Play / Stop |
| `0` (pavé num.) | RAM Preview |
| `Home` / `End` | Début / Fin de la timeline |
| `Page Up` / `Page Down` | Image précédente / suivante |
| `I` / `O` | Aller au In Point / Out Point du calque sélectionné |
| `B` / `N` | Définir le début / la fin de la zone de travail |

---

## Calques

| Raccourci | Action |
|---|---|
| `Ctrl/Cmd + Y` | Créer un solide |
| `Ctrl/Cmd + Alt + Y` | Créer un calque d'ajustement |
| `Ctrl/Cmd + D` | Dupliquer le calque |
| `[` / `]` | Déplacer le In Point / Out Point au CTI |
| `Alt + [` / `Alt + ]` | Trimmer le calque au CTI |
| `Ctrl/Cmd + Maj + D` | Couper le calque au CTI (Split layer) |
| `Ctrl/Cmd + Maj + C` | Précomposer les calques sélectionnés |
| `F4` | Basculer l'affichage Modes / Parent dans la timeline |

---

## Propriétés dans la timeline

| Raccourci | Action |
|---|---|
| `P` | Position |
| `S` | Scale |
| `R` | Rotation |
| `T` | Opacity (transparence) |
| `A` | Anchor Point |
| `E` | Effets |
| `U` | Propriétés avec keyframes |
| `UU` (double frappe) | Propriétés modifiées par rapport au défaut |
| `Alt + clic sur chrono` | Ajouter une expression sur la propriété |

---

## Keyframes

| Raccourci | Action |
|---|---|
| `Alt + P/S/R/T/A` | Créer un keyframe sur la propriété |
| `F9` | Easy Ease sur la sélection |
| `Maj + F9` | Easy Ease In |
| `Ctrl/Cmd + Maj + F9` | Easy Ease Out |
| `K` / `J` | Keyframe suivant / précédent |

---

## Composition et rendu

| Raccourci | Action |
|---|---|
| `Ctrl/Cmd + N` | Nouvelle composition |
| `Ctrl/Cmd + K` | Réglages de composition |
| `Ctrl/Cmd + M` | Ajouter à la Render Queue |
| `Ctrl/Cmd + Alt + M` | Ajouter à Adobe Media Encoder |

---

## Vue et outils

| Raccourci | Action |
|---|---|
| `,` / `.` | Zoom arrière / avant dans le panneau Comp |
| `Maj + /` | Ajuster la comp à la fenêtre (Fit) |
| `V` | Outil sélection |
| `W` | Outil rotation |
| `Z` | Outil zoom |
| `H` | Outil main (pan) |
| `~` (tilde) | Maximiser le panneau sous le curseur |

---

## Astuces de saisie rapide

- **Maj + glisser** une valeur numérique → incréments de ×10
- **Ctrl/Cmd + glisser** une valeur → incréments de ×0.1
- **Double-cliquer** sur une valeur → saisie directe au clavier
- **Clic droit sur le chrono** d'une propriété → toutes les options de keyframe
