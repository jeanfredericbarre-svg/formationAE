# La famille complète des méthodes de conversion

AE fournit quatre méthodes pour convertir des coordonnées entre référentiels. Elles existent par paires — chaque méthode a son inverse.

| Méthode | Direction | Référentiel source | Référentiel cible |
|---|---|---|---|
| `toComp(point)` | → | Local du calque | Composition |
| `fromComp(point)` | ← | Composition | Local du calque |
| `toWorld(point)` | → | Local du calque | World (3D) |
| `fromWorld(point)` | ← | World (3D) | Local du calque |

En **2D**, `toComp` et `toWorld` retournent des résultats identiques — la composition et le world sont le même référentiel. La distinction n'existe vraiment qu'en **3D**, quand des calques ont des orientations différentes dans l'espace.

---

# toWorld() et fromWorld()

`toWorld()` et `fromWorld()` opèrent dans l'espace **3D global** de la scène — indépendamment de toute caméra ou point de vue.

```javascript
thisLayer.toWorld([0, 0, 0])
// → position du pivot du calque dans l'espace 3D mondial
//   retourne un Array [x, y, z]
```

Ces méthodes sont particulièrement utiles quand on travaille avec des calques 3D qui ont des rotations dans les 3 axes — le world space garantit des coordonnées absolues, non affectées par les rotations des parents.

```javascript
// Trouver la position 3D réelle d'un point quelconque d'un calque :
thisLayer.toWorld([100, 50, 0])
// → les coordonnées mondiales du point situé à [100, 50] en local sur ce calque
```

# toComp() et fromComp()

`toComp()` et `fromComp()` opèrent dans l'espace **2D de la composition** — c'est-à-dire les coordonnées pixels telles qu'elles apparaissent à l'écran.

```javascript
thisLayer.toComp([0, 0])
// → position du pivot du calque en coordonnées de composition
//   retourne un Array [x, y]
```

C'est la paire la plus utilisée en 2D pour faire communiquer des calques qui ont des parents différents.

**Exemple — faire pointer un calque vers un autre :**

```javascript
// Sur la propriété Rotation du calque "Flèche" :
var cible = thisComp.layer("Cible").toComp([0, 0]);
var source = thisLayer.toComp([0, 0]);

var dx = cible[0] - source[0];
var dy = cible[1] - source[1];

radiansToDegrees(Math.atan2(dy, dx))
// → calcule l'angle entre les deux calques et l'applique à la rotation
```

# Cas pratiques

## Attacher un calque au coin d'un autre calque

```javascript
// Sur la propriété Position du calque "Badge" :
var hote = thisComp.layer("Carte");

// Coin supérieur droit de "Carte" en local space :
var coinLocal = [hote.width / 2, -hote.height / 2];

// Convertir en coordonnées de composition :
var coinComp = hote.toComp(coinLocal);

// Convertir en local space du calque "Badge" :
thisLayer.fromComp(coinComp)
```

Le Badge se colle au coin de la Carte — et si la Carte bouge, tourne ou change de parent, le Badge suit automatiquement.

## Connecter deux calques avec une ligne

Pour calculer la longueur et l'angle entre deux calques (utile pour un calque Shape ou un masque) :

```javascript
// Position des deux calques en espace composition :
var p1 = thisComp.layer("Point A").toComp([0, 0]);
var p2 = thisComp.layer("Point B").toComp([0, 0]);

var dx = p2[0] - p1[0];
var dy = p2[1] - p1[1];

// Longueur de la ligne :
var longueur = Math.sqrt(dx * dx + dy * dy);

// Angle :
var angle = radiansToDegrees(Math.atan2(dy, dx));
```

> 💡 `Math.atan2(dy, dx)` est une fonction JavaScript standard — elle retourne l'angle en **radians** entre deux points. `radiansToDegrees()` est une méthode propre à AE qui convertit ce résultat en degrés, utilisable directement comme valeur de Rotation.
