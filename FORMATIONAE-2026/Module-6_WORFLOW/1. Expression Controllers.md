# Pourquoi des Expression Controllers ?

On a vu dans les modules précédents qu'une expression peut lire la valeur d'une propriété d'un autre calque. Mais piloter une rotation depuis la position X d'un calque est peu intuitif pour quelqu'un qui doit modifier l'animation après coup — il faut savoir où regarder et comprendre l'expression.

Les **Expression Controllers** résolvent ce problème. Ce sont des effets AE dont le seul rôle est d'**exposer une valeur nommée et accessible**, sans aucun effet visuel. Ils servent de **tableau de bord** pour toute la composition.

> 💡 Analogie : les Expression Controllers, c'est le tableau de bord d'un avion. Le pilote n'a pas besoin d'accéder aux câbles dans la carlingue — il tourne un bouton sur le tableau de bord, et les systèmes répondent. Les expressions sont les câbles ; les controllers sont les boutons.

**Où les trouver :** Effets → Contrôles d'expression → (choisir le type)

# Les 5 types de controllers

## Slider Control (Curseur)

Expose un **nombre**. Plage par défaut : 0 à 100, mais modifiable.

```javascript
// Lire un Slider depuis une expression :
effect("Vitesse")("Curseur")
// ou depuis un autre calque :
thisComp.layer("CONTROLS").effect("Vitesse")("Curseur")
```

Usages : vitesses, durées, distances, intensités, nombre de copies.

## Angle Control (Angle)

Expose un **angle en degrés**. Interface avec molette rotative dans le panneau Effets.

```javascript
effect("Direction")("Angle")
```

Usages : rotations, orientations, direction d'un mouvement.

## Checkbox Control (Case à cocher)

Expose un **booléen** — retourne 0 (décoché) ou 1 (coché).

```javascript
// Afficher/masquer conditionnellement :
if (effect("Visible")("Case à cocher") === 1) {
  100   // opacité 100%
} else {
  0     // opacité 0%
}
```

Usages : afficher/masquer des éléments, activer/désactiver des comportements.

## Color Control (Couleur)

Expose une **couleur** au format Array `[R, G, B, A]` avec des valeurs entre 0 et 1.

```javascript
effect("Couleur principale")("Couleur")
// → [0.2, 0.6, 1, 1] par exemple (bleu clair)
```

Usages : piloter la couleur de plusieurs éléments depuis un seul point de contrôle.

## Point Control (Point)

Expose une **position 2D** — interface avec une croix déplaçable dans la fenêtre de composition.

```javascript
effect("Point de départ")("Point")
// → [960, 540] par exemple
```

Usages : définir un point d'attraction, une direction, un repère dans la composition.

# Mettre en place un système de contrôle

**Étape 1 — Créer le calque CONTROLS**

Calque → Nouveau → Objet nul → renommer "CONTROLS". Ce calque est nul (invisible au rendu) et sert uniquement à porter les effets.

**Étape 2 — Ajouter les controllers nécessaires**

Sur le calque CONTROLS : Effets → Contrôles d'expression → Curseur (ou autre type). Répéter pour chaque paramètre à contrôler.

**Étape 3 — Nommer les controllers**

Dans la timeline, double-cliquer sur le nom de chaque effet pour le renommer. Le nom est ce qui apparaît dans l'expression — un nom explicite évite les erreurs.

**Étape 4 — Écrire les expressions**

Sur chaque propriété à piloter, attacher une expression qui va lire la valeur du controller :

```javascript
// Sur la Rotation de plusieurs calques :
var ctrl = thisComp.layer("CONTROLS").effect("Vitesse rotation")("Curseur");
ctrl * time
```

**Étape 5 — Verrouiller et étiqueter**

Verrouille le calque CONTROLS (icône cadenas) pour éviter de le déplacer accidentellement. Donne-lui une couleur d'étiquette distinctive dans la timeline.

# Nommer et organiser ses controllers

Avec un projet complexe, les controllers prolifèrent vite. Quelques conventions utiles :

**Nommage par section :**
```
"[Intro] Durée"
"[Intro] Couleur fond"
"[Logo] Vitesse rotation"
"[Texte] Délai entre lettres"
```

**Regrouper par calque CONTROLS dédié :**

Pour des projets longs, plusieurs calques CONTROLS thématiques valent mieux qu'un seul surchargé :
```
CONTROLS_COULEURS
CONTROLS_TIMING
CONTROLS_POSITIONS
```

> 💡 Un système de controllers bien nommé transforme une composition complexe en quelque chose qu'un autre animateur peut modifier sans lire une seule expression. C'est la marque d'un travail professionnel et livrable.
