## sourceText : piloter le contenu d'un calque texte

`sourceText` est la propriété qui contient le **texte affiché** par un calque texte. Comme toute propriété animable, on peut lui attacher une expression — et ainsi **piloter dynamiquement ce qui est écrit**, sans avoir à modifier le calque manuellement.

**Pour y accéder :** déplie le calque texte → **Texte** → **Texte source** → `Alt+clic` sur le chronomètre pour activer l'expression.

L'expression doit retourner une **String** (chaîne de caractères).

```javascript
// Expression la plus simple — du texte statique via expression :
"Bonjour"

// Plus utile : du texte dynamique construit par calcul
"Frame : " + timeToFrames(time)
```

> 💡 `sourceText` est particulier : il accepte à la fois une String simple et un objet texte enrichi (avec style). Dans la grande majorité des cas, une String suffit.

## Afficher time en tant que timecode

Un cas d'usage classique : un calque texte qui affiche le timecode courant, mis à jour à chaque frame automatiquement.

**`timeToFrames(t)`** convertit un temps en secondes en numéro de frame :
```javascript
timeToFrames(time)          // frame courante selon le framerate de la comp
timeToFrames(time, 25)      // force le calcul en 25fps, quelle que soit la comp
```

**`framesToTime(n)`** fait l'inverse — convertit un numéro de frame en secondes.

**`timeToTimecode(t)`** retourne directement une String au format `HH:MM:SS:FF` :
```javascript
// Sur sourceText :
timeToTimecode(time)        // "00:00:02:14" par exemple
```

**Exemple — afficher la frame courante :**
```javascript
"Frame : " + Math.floor(timeToFrames(time))
// → "Frame : 0", "Frame : 1", "Frame : 2"...
```

## Lier le texte à un autre calque

`sourceText` peut aller lire le contenu d'un autre calque texte — ce qui permet de **centraliser un texte** et de le répercuter automatiquement sur plusieurs calques.

```javascript
thisComp.layer("Titre principal").text.sourceText
// Ce calque affiche exactement ce qu'affiche "Titre principal"
// Modifie "Titre principal" → tous les calques liés se mettent à jour
```

> 💡 Utile pour des génériques où le même titre apparaît à plusieurs endroits dans la composition, ou dans des précompositions différentes.

## Construire des chaînes dynamiques

En JavaScript, l'opérateur `+` concatène des chaînes. On peut ainsi construire des textes dynamiques en combinant des valeurs fixes et des valeurs calculées.

```javascript
// Afficher la durée restante avant la fin du calque :
var restant = outPoint - time;
var secondes = Math.floor(restant);
"Fin dans : " + secondes + "s"
```

**Convertir un nombre en String propre :**

```javascript
// Math.floor() supprime les décimales
// toFixed(n) arrondit à n décimales et retourne une String
var val = 3.14159;
val.toFixed(2)     // → "3.14"
```

**Exemple complet — un compteur qui descend :**

```javascript
var duree = outPoint - inPoint;
var progression = (time - inPoint) / duree;
var compte = Math.ceil((1 - progression) * 10);  // compte de 10 à 1
Math.max(compte, 0) + ""                          // évite les valeurs négatives, force String
```

> 💡 N'oublie pas que `sourceText` attend une **String**. Si ton expression retourne un `Number`, AE peut afficher une erreur. Ajoute `+ ""` à la fin pour forcer la conversion en chaîne de caractères.
