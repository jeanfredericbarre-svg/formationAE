<span id="limite"></span>
## La limite des Text Animators sans expression

Un Text Animator standard fonctionne bien pour des animations prévisibles et répétitives : entrée lettre par lettre, révélation par Range Selector, effet de rebond sur les caractères. Mais dès que l'animation doit **réagir à quelque chose** — la position d'un objet, le tempo d'une musique, un événement dans la timeline — les keyframes atteignent leur limite.

Le problème fondamental : une keyframe encode une valeur à un instant donné. Elle ne peut pas lire l'état d'un autre calque, ni réagir à une condition externe. Combiner un Text Animator avec un Expression Selector ouvre cette possibilité : **l'intensité appliquée à chaque caractère devient le résultat d'un calcul**, pas d'une interpolation entre deux keyframes.

<span id="zone"></span>
## Zone d'influence spatiale : lier l'intensité à la position d'un calque

L'idée : un calque Null se déplace dans la composition, et les caractères du texte sont affectés par l'animateur en fonction de leur proximité avec ce Null. Les lettres dans la zone d'influence sont pleinement affectées, les lettres éloignées ne le sont pas.

**Mise en place :**
- Un calque Null nommé `CURSEUR` positionné librement dans la comp
- Un Text Animator sur le calque texte (Opacité, Echelle, ou Position)
- Un Expression Selector dont l'intensité est calculée depuis la distance au Null

```javascript
// Expression Selector — zone d'influence spatiale
const curseur = thisComp.layer("CURSEUR").transform.position;

// Position approximative du caractère courant dans la comp
// textIndex va de 1 à textTotal — on répartit les caractères sur la largeur du calque
const largeur = thisLayer.width;
const xCaractere = thisLayer.transform.position[0] - largeur / 2
                 + (textIndex / textTotal) * largeur;
const yCaractere = thisLayer.transform.position[1];

// Distance entre le curseur et le caractère
const dx = curseur[0] - xCaractere;
const dy = curseur[1] - yCaractere;
const distance = Math.sqrt(dx * dx + dy * dy);

// Rayon d'influence (en pixels)
const rayon = 150;

// Convertir la distance en intensité 0-100
const intensite = linear(distance, 0, rayon, 100, 0);
Math.max(0, intensite)
```

> 💡 La position par caractère est ici une approximation linéaire : les caractères sont répartis uniformément sur la largeur du calque. Pour un résultat pixel-perfect, il faudrait passer par un script ExtendScript qui lit les positions réelles via `sourceRectAtTime`. L'approximation est suffisante pour la grande majorité des effets visuels.

Animer la position du Null `CURSEUR` dans la timeline fait "balayer" la zone d'influence sur le texte.

<span id="marqueurs"></span>
## Synchroniser sur des marqueurs de calque

Les marqueurs AE permettent de poser des repères temporels sur un calque. Via expression, il est possible de lire ces marqueurs et de déclencher ou moduler un effet au moment où la tête de lecture les franchit.

**Accéder aux marqueurs par index :**

```javascript
// Temps du premier marqueur du calque son
const calqueSon = thisComp.layer("SON");
const tempsMarqueur = calqueSon.markerKey(1).time
```

**Construire une intensité qui pulse à chaque marqueur :**

```javascript
// Pulse à chaque marqueur — décroissance rapide après chaque impact
const calqueSon = thisComp.layer("SON");
const nbMarqueurs = calqueSon.marker.numKeys;

let intensiteMax = 0;

for (let i = 1; i <= nbMarqueurs; i++) {
    const tMarqueur = calqueSon.markerKey(i).time;
    const delta = time - tMarqueur;

    // Ne considérer que les marqueurs déjà passés
    if (delta >= 0) {
        // Décroissance exponentielle après le marqueur
        const pulse = Math.exp(-delta * 8) * 100;
        if (pulse > intensiteMax) intensiteMax = pulse;
    }
}

intensiteMax
```

Chaque fois que la tête de lecture dépasse un marqueur, l'intensité monte à 100 puis décroît rapidement. La vitesse de décroissance est contrôlée par le coefficient `8` dans `Math.exp(-delta * 8)` : plus il est élevé, plus la décroissance est rapide.

> 💡 Cette technique est particulièrement efficace pour synchroniser une animation texte sur les beats d'une musique : poser les marqueurs manuellement sur les temps forts, puis laisser l'expression faire le reste.

<span id="audio"></span>
## Réagir au niveau audio

Pour aller plus loin que les marqueurs, il est possible de lire le niveau audio frame par frame et de l'utiliser directement comme intensité du sélecteur.

```javascript
// Intensité pilotée par le niveau audio en temps réel
const calqueSon = thisComp.layer("SON");
const niveau = calqueSon.audioLevels[0];  // canal gauche, en dB

// Convertir les dB (valeurs négatives) en intensité 0-100
// Un niveau de -40 dB est considéré comme silence, 0 dB comme maximum
const intensite = linear(niveau, -40, 0, 0, 100);
Math.max(0, Math.min(100, intensite))
```

Le résultat est une intensité qui suit fidèlement le volume de la piste audio. Pour calibrer la sensibilité, remplacer `-40` par une valeur plus haute (ex. `-20`) pour que l'effet se déclenche sur des sons moins forts.

<span id="pratique"></span>
## Cas pratique : un lower third piloté par expressions

Un lower third (bandeau d'identification) avec un nom et un titre, dont l'entrée et la sortie des caractères sont pilotées par des expressions et un Slider de contrôle.

**Structure de la comp :**

```
Null "CTRL_LOWER"
  ├── Curseur "Entree"   (0 = hors champ, 100 = pleinement visible)
  └── Curseur "Sortie"   (0 = visible, 100 = sorti)

Calque texte "NOM"
  └── Animateur : Opacité + Position Y
      └── Expression Selector

Calque texte "TITRE"
  └── Animateur : Opacité + Position Y
      └── Expression Selector (même logique, délai décalé)
```

**Expression Selector sur le calque "NOM" :**

```javascript
// Entrée séquentielle de gauche à droite, sortie de droite à gauche
const ctrl = thisComp.layer("CTRL_LOWER");
const entree = ctrl.effect("Entree")("Curseur");
const sortie = ctrl.effect("Sortie")("Curseur");

// Délai par caractère pour l'entrée
const delaiEntree = (textIndex / textTotal) * 30;
const intensiteEntree = Math.max(0, entree - delaiEntree);

// Délai par caractère inversé pour la sortie (de droite à gauche)
const delaiSortie = ((textTotal - textIndex) / textTotal) * 30;
const intensiteSortie = Math.max(0, sortie - delaiSortie);

// Combiner : l'entrée augmente l'intensité, la sortie la diminue
Math.min(100, Math.max(0, intensiteEntree - intensiteSortie))
```

**Valeurs de l'animateur :**
- Opacité : `0%` (les caractères partent de transparent)
- Position Y : `20` (les caractères montent de 20px lors de l'entrée)

**Utilisation :**
- Animer le Slider `Entree` de `0` à `100` sur 15 frames : les lettres du nom entrent séquentiellement de gauche à droite
- Le calque `TITRE` utilise la même expression avec un léger décalage temporel sur le Slider `Entree` (démarrer 5 frames après le nom)
- Animer le Slider `Sortie` de `0` à `100` pour la sortie : les lettres partent de droite à gauche

Ce rig est entièrement réutilisable : changer le texte des calques ne nécessite aucune retouche des expressions ni des Sliders.
