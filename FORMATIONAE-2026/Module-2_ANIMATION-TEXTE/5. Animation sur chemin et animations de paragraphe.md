## Texte sur un tracé

After Effects permet d'associer un calque texte à un **tracé de forme** (*path*) pour que les caractères suivent une courbe, un cercle, ou n'importe quelle trajectoire dessinée.

Cette technique est particulièrement utile pour :
- les titres cinématiques courbés,
- les textes circulaires (badges, logos animés),
- les animations où le texte épouse une trajectoire narrative.

### Créer un texte sur tracé

1. Sélectionne ton calque texte dans la timeline.
2. Avec l'outil **Plume** (G), dessine un tracé directement sur le calque texte (pas sur un calque de forme séparé). Un masque est créé automatiquement.
3. Dans les propriétés du calque texte, ouvre **Texte → Options de tracé**.
4. Dans le menu **Tracé**, sélectionne le masque que tu viens de créer (**Masque 1** par défaut).

> 💡 Le tracé utilisé est un **masque** du calque texte — pas un tracé de calque de forme. C'est une distinction importante : si tu crées le tracé sur un calque de forme séparé, il ne sera pas disponible dans le menu Tracé.

---

<span id="parametres-path"></span>
## Paramètres du tracé

Une fois le tracé sélectionné, quatre paramètres apparaissent dans **Options de tracé** :

| Paramètre | Rôle |
|---|---|
| **Inverser le tracé** | Retourne le sens de lecture du texte sur le tracé |
| **Perpendiculaire au tracé** | Oriente chaque caractère perpendiculairement à la tangente du tracé |
| **Forcer l'alignement** | Étire ou contracte les espacements pour que le texte occupe tout le tracé |
| **Marge de début / Marge de fin** | Déplace le point de départ et d'arrivée du texte sur le tracé |

> 💡 **Marge de début** est le paramètre clé pour les animations : en le keyframant, tu fais glisser le texte le long du tracé. C'est l'équivalent d'un *offset* sur la trajectoire.

<span id="animer-path"></span>
## Animer le texte sur le tracé

### Défilement le long du tracé

La méthode la plus directe :

1. Place un keyframe sur **Marge de début** à `0` à l'instant de départ.
2. Place un second keyframe à la valeur souhaitée (par exemple `500 px` ou davantage selon la longueur du tracé) à l'instant d'arrivée.
3. Applique une interpolation **Easy Ease** (F9) pour adoucir le mouvement.

```js
// Expression pour un défilement continu automatique
linear(time, 0, 10, 0, 2000)
```

Ce code fait défiler le texte de `0` à `2000 px` sur 10 secondes de façon linéaire.

### Faire apparaître le texte lettre par lettre

Combine le path-based avec un **Text Animator** sur l'opacité :

1. Dans **Texte → Animer**, ajoute **Opacité**.
2. Dans le **Range Selector**, keyframe **Décalage** de `-100 %` à `0 %`.
3. Les caractères apparaissent progressivement tout en suivant le tracé.

> 💡 Le Text Animator s'applique sur les caractères **dans leur position sur le tracé**, les deux systèmes sont parfaitement compatibles et se combinent naturellement.

### Texte circulaire animé

Pour un texte sur un cercle (badge, timecode circulaire) :

1. Crée un masque **ellipse** sur le calque texte (double-clic sur l'outil Ellipse avec le calque texte sélectionné).
2. Active **Forcer l'alignement** pour répartir les caractères uniformément.
3. Keyframe **Marge de début** pour faire tourner le texte autour du cercle.

## Tracking et leading animés

Le **tracking** (espacement horizontal entre les caractères) et le **leading** (interlignage vertical) sont deux propriétés de paragraphe animables directement via les **Text Animators**.

### Animer le tracking

1. Dans **Texte → Animer**, ajoute **Espacement des caractères** (*Tracking Amount*).
2. Un *Animator* est créé avec un *Range Selector* par défaut.
3. Keyframe la valeur de tracking : une valeur négative resserre les caractères, une valeur positive les écarte.

```
Tracking = 0   → espacement normal
Tracking = 50  → caractères très écartés
Tracking = -20 → caractères resserrés (attention aux collisions)
```

**Cas d'usage typique : reveal par expansion**

| Temps | Tracking | Opacité |
|---|---|---|
| 0 s | 80 | 0 % |
| 0,5 s | 0 | 100 % |

Les lettres arrivent en se resserrant tout en apparaissant, un classique du motion design de titrage.

> 💡 Pour que l'animation parte du centre, règle l'**alignement** du *Range Selector* sur **Centre** dans ses options avancées.

### Animer le leading

Le leading (interlignage) est animable de la même façon :

1. Dans **Texte → Animer**, ajoute **Interlignage** (*Line Anchor* + *Leading*).
2. Keyframe la valeur pour faire apparaître ou disparaître des lignes de texte en les faisant glisser verticalement.

**Cas d'usage typique : révélation de lignes**

Un bloc de trois lignes part avec un leading très négatif (les lignes se superposent) et s'ouvre progressivement jusqu'à l'interlignage normal. Combiné à un masque de découpe sur le calque, cela donne l'illusion que les lignes émergent une à une.

```
Leading = -60 px  → lignes empilées
Leading =   0 px  → interlignage normal
```

### Combiner tracking et leading

Les deux animators peuvent coexister sur le même calque texte. Chacun possède son propre *Range Selector*, ce qui permet des timings indépendants :

- Le tracking s'anime de `80` à `0` entre 0 s et 0,4 s.
- Le leading s'anime de `-40` à `0` entre 0,1 s et 0,5 s avec un léger décalage.

Le résultat : le texte se déploie à la fois horizontalement et verticalement, avec une dynamique plus riche qu'une simple apparition en fondu.

<span id="exercice"></span>
## Exercice pratique

### Objectif

Créer un titre animé en deux temps :
1. Un texte court suit un tracé courbé en entrant dans le cadre.
2. Une fois en place, le tracking s'ouvre de `60` à `0` pour révéler les caractères.

### Étapes

1. Crée une composition `1920×1080`, 5 secondes, 25 fps.
2. Ajoute un calque texte avec le mot **CINÉMA** en majuscules, taille `120 px`.
3. Dessine un tracé légèrement courbé (arc doux) avec la **Plume** sur ce calque.
4. Dans **Options de tracé**, sélectionne **Masque 1**.
5. Keyframe **Marge de début** : `-600 px` à `0 s` → `0 px` à `1 s`.
6. Ajoute un **Text Animator** sur **Opacité** + **Espacement des caractères**.
7. À `0,8 s` : Opacité `0`, Tracking `60`. À `1,5 s` : Opacité `100 %`, Tracking `0`.
8. Applique **Easy Ease** sur tous les keyframes.

> 💡 L'entrée sur le tracé et l'ouverture du tracking se chevauchent légèrement dans le temps, c'est ce chevauchement qui donne de la fluidité et de la personnalité à l'animation.
