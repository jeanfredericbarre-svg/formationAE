## Accéder au tracking et au leading via expression

Le tracking et le leading ne font pas partie des propriétés de transformation standard d'un calque. Ils appartiennent au **moteur typographique** d'AE et s'accèdent via le chemin de propriété `text.animator`.

Pour les atteindre, la méthode la plus fiable est de créer un **Text Animator dédié** contenant la propriété souhaitée, puis de la référencer via expression depuis un autre calque, ou d'y attacher directement une expression.

**Tracking :** dans AE, le tracking global d'un calque texte s'appelle **Espacement des caractères** (*Tracking Amount*). Il est accessible via :

```javascript
// Attacher directement une expression sur "Espacement des caractères"
// dans un Text Animator — Alt+clic sur le chronomètre de la propriété
const progression = thisComp.layer("CTRL").effect("Progression")("Curseur");
linear(progression, 0, 100, 200, 0)
// à progression 0 : tracking à 200 (lettres très espacées)
// à progression 100 : tracking à 0 (espacement normal)
```

**Leading :** le leading (*Interlignage*) s'accède via la propriété **Interlignage automatique** ou **Interlignage** dans un animateur texte, selon le type de calque (point ou paragraphe).

> 💡 Ces propriétés n'apparaissent dans un animateur que si tu les as explicitement ajoutées via **Animateur → Ajouter → Propriété → Espacement des caractères** (ou Interlignage). Elles ne sont pas présentes par défaut comme l'Opacité ou la Position.

## Tracking animé : respiration typographique

Un `Math.sin` sur la propriété **Espacement des caractères** d'un animateur produit une compression et expansion rythmique du texte, sans aucune keyframe.

```javascript
// Expression sur "Espacement des caractères" dans un Text Animator
const vitesse = 0.8;       // fréquence de l'oscillation
const amplitude = 30;      // variation max en unités de tracking

const sinus = Math.sin(time * vitesse * Math.PI * 2);
sinus * amplitude
// résultat : oscille entre -30 et +30 autour de la valeur de base
```

Pour piloter l'amplitude depuis un Slider :

```javascript
const ctrl = thisComp.layer("CTRL_TEXTE");
const amp = ctrl.effect("Amplitude")("Curseur");
const vitesse = ctrl.effect("Vitesse")("Curseur") * 0.1;

const sinus = Math.sin(time * vitesse * Math.PI * 2);
sinus * amp
```

> 💡 Le tracking en AE est exprimé en **millièmes de cadratin** (*em units* / 1000). Une valeur de `100` représente un dixième de la taille de la police. Sur un texte à 100px, `100` de tracking ajoute 10px d'espacement entre chaque caractère.

## Leading dynamique : adapter l'interlignage au contenu

Quand un calque texte affiche un contenu dynamique (piloté par `sourceText`), le nombre de lignes peut varier. Un leading fixe produit alors des résultats incohérents selon le contenu. Une expression permet de **calculer un interlignage proportionnel** à la taille de la police ou au nombre de lignes.

**Interlignage proportionnel à la taille de la police :**

```javascript
// Expression sur la propriété "Interlignage" d'un Text Animator
// Récupérer la taille de police du calque (via un Slider de contrôle)
const taillePolice = thisComp.layer("CTRL_TEXTE").effect("Taille police")("Curseur");
taillePolice * 1.4
// interlignage = 140% de la taille — valeur typographique standard
```

**Interlignage qui s'adapte au nombre de lignes :**

```javascript
// Compresser l'interlignage si le texte est long
const nbLignes = thisLayer.text.sourceText.toString().split("\n").length;
const baseLeading = 60;
const compression = Math.max(0.7, 1 - (nbLignes - 1) * 0.1);
baseLeading * compression
// 1 ligne : leading normal
// 2 lignes : leading à 90%
// 4 lignes : leading à 70% (plancher)
```

## Combiner tracking et leading sur un même calque

Tracking et leading peuvent être animés simultanément depuis un même Slider `Progression`, créant un effet de **déploiement typographique** où le texte s'ouvre à la fois horizontalement et verticalement.

**Structure :**
- Un seul Slider `Progression` (0 à 100) sur un Null `CTRL_TEXTE`
- Un Text Animator avec deux propriétés : Espacement des caractères + Interlignage
- Deux expressions distinctes sur chaque propriété, lisant le même Slider

```javascript
// Expression sur "Espacement des caractères"
const progression = thisComp.layer("CTRL_TEXTE").effect("Progression")("Curseur");
linear(progression, 0, 100, 150, 0)
// tracking part de 150, converge vers 0
```

```javascript
// Expression sur "Interlignage"
const progression = thisComp.layer("CTRL_TEXTE").effect("Progression")("Curseur");
linear(progression, 0, 100, 120, 0)
// leading part de 120 unités supplémentaires, converge vers 0
```

Animer le Slider de 0 à 100 produit une convergence simultanée du tracking et du leading : le texte "se referme" sur sa forme finale.

> 💡 En ajoutant une propriété Opacité au même animateur (valeur de départ à `0%`), le texte apparaît en même temps qu'il se contracte — les trois propriétés sont pilotées par le même Slider, en une seule keyframe.

<span id="pratique"></span>
## Cas pratique : un titre qui se contracte à l'entrée

**Objectif :** un titre dont les lettres partent très espacées et transparentes, puis convergent vers leur position finale en fondu, piloté par un seul Slider `Progression`.

**Structure du rig :**

```
Null "CTRL_TITRE"
  └── Curseur "Progression"   (0 = départ, 100 = position finale)

Calque texte "TITRE"
  └── Animateur "ENTREE"
      ├── Opacité
      ├── Espacement des caractères
      └── Interlignage
          └── Range Selector (Début 0%, Fin 100% — affecte tous les caractères)
```

**Expression sur Opacité :**

```javascript
const progression = thisComp.layer("CTRL_TITRE").effect("Progression")("Curseur");
linear(progression, 0, 60, 0, 100)
// l'opacité atteint 100% dès que Progression dépasse 60
// le reste du mouvement (60 à 100) est purement typographique
```

**Expression sur Espacement des caractères :**

```javascript
const progression = thisComp.layer("CTRL_TITRE").effect("Progression")("Curseur");
const easeVal = ease(progression, 0, 100, 200, 0);
easeVal
// ease() produit une décélération naturelle — les lettres ralentissent en arrivant
```

**Expression sur Interlignage :**

```javascript
const progression = thisComp.layer("CTRL_TITRE").effect("Progression")("Curseur");
ease(progression, 0, 100, 80, 0)
// interlignage supplémentaire de 80 unités au départ, converge vers 0
```

**Utilisation :**
- Animer le Slider `Progression` de `0` à `100` sur 20 frames avec une courbe Ease Out
- L'opacité monte rapidement sur les 12 premières frames
- Le tracking et le leading convergent sur toute la durée avec une décélération naturelle

Pour une **sortie**, dupliquer le Null et l'animateur, inverser les valeurs de départ et d'arrivée dans les expressions, et animer le Slider `Progression` dans l'autre sens.
