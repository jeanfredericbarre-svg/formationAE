## selectorValue : la vraie variable de travail

Dans un **Expression Selector**, la variable `selectorValue` représente l'**intensité appliquée au caractère courant** par le sélecteur, sous forme d'un tableau `[valeur]` compris entre `0` et `100` (et non entre 0 et 1 comme on pourrait s'y attendre).

C'est la distinction fondamentale avec `textIndex` :

| Variable | Ce qu'elle retourne |
|---|---|
| `textIndex` | La position du caractère dans la chaîne (1, 2, 3…) |
| `textTotal` | Le nombre total de caractères |
| `selectorValue` | L'intensité courante appliquée à ce caractère par le sélecteur (0 à 100) |

`selectorValue` est ce que l'animateur va réellement utiliser pour calculer son effet. En retournant une valeur modifiée, tu **remplaces** ce que le sélecteur aurait produit par défaut.

```javascript
// Retourner selectorValue tel quel — comportement identique à un Range Selector standard
selectorValue
```

```javascript
// Inverser l'intensité
const valeur = selectorValue[0];
100 - valeur
```

> 💡 `selectorValue` est un tableau à une dimension : utilise `selectorValue[0]` pour accéder à la valeur numérique dans les calculs.

## Vague et pulsation avec Math.sin

`Math.sin` produit une oscillation régulière entre -1 et 1. Appliqué à chaque caractère avec un décalage basé sur `textIndex`, il génère un **effet de vague** qui se propage le long du texte.

```javascript
// Vague progressive sur les caractères
const frequence = 2;       // nombre d'ondulations visibles
const vitesse = 1.5;       // vitesse de défilement
const decalage = textIndex * 0.4;  // décalage de phase entre les lettres

const sinus = Math.sin(time * vitesse * Math.PI * 2 + decalage * frequence);

// Convertir de [-1, 1] vers [0, 100]
(sinus + 1) / 2 * 100
```

Pour un effet de **pulsation simultanée** (toutes les lettres bougent ensemble, sans décalage) :

```javascript
// Pulsation globale — toutes les lettres à l'unisson
const vitesse = 1;
const sinus = Math.sin(time * vitesse * Math.PI * 2);
(sinus + 1) / 2 * 100
```

> 💡 Le terme `decalage * frequence` dans le `Math.sin` est la **phase** de chaque caractère. Plus `decalage` est grand, plus les lettres sont déphasées les unes par rapport aux autres, rendant la vague plus prononcée.

## Bruit organique avec wiggle

`wiggle` dans un Expression Selector produit une agitation indépendante sur chaque caractère, avec un résultat aléatoire mais contrôlé.

```javascript
// Agitation aléatoire par caractère
const freq = 2;       // fréquence des variations (Hz)
const amp  = 50;      // amplitude (0-100)

// Le seed basé sur textIndex garantit que chaque lettre a son propre mouvement
wiggle(freq, amp, 1, 0.5, time + textIndex * 100)[0]
```

Le quatrième argument de `wiggle` est le **seed** — en le liant à `textIndex`, chaque lettre reçoit une graine différente et donc un mouvement indépendant.

Pour combiner vague et bruit :

```javascript
// Vague de base + perturbation organique
const base = Math.sin(time * 2 + textIndex * 0.5) * 50 + 50;
const bruit = wiggle(3, 15, 1, 0.5, time + textIndex * 100)[0];
Math.min(100, Math.max(0, base + bruit - 50))
```

## Coupler à un Slider Controller

Piloter l'intensité ou la fréquence depuis un **Slider** placé sur un calque Null permet de contrôler l'animation depuis la timeline, sans rouvrir l'expression.

**Mise en place :**
1. Créer un calque Null nommé `CTRL_TEXTE`
2. Lui ajouter l'effet **Curseur** (`Effet → Expression Controls → Curseur`)
3. Renommer le curseur `Intensite` (double-clic sur le nom de l'effet)
4. Dans l'Expression Selector du calque texte, référencer ce curseur :

```javascript
// Récupérer la valeur du Slider sur le Null de contrôle
const ctrl = thisComp.layer("CTRL_TEXTE").effect("Intensite")("Curseur");

// Utiliser cette valeur comme amplitude de la vague
const amp = ctrl;   // le Slider va de 0 à 100 par défaut
const sinus = Math.sin(time * 2 + textIndex * 0.5);
(sinus + 1) / 2 * amp
```

Animer le Slider de 0 à 100 dans la timeline produit un fondu d'entrée progressif de l'effet, sans toucher à l'expression.

Pour contrôler aussi la **fréquence** depuis un second Slider :

```javascript
const amp  = thisComp.layer("CTRL_TEXTE").effect("Intensite")("Curseur");
const freq = thisComp.layer("CTRL_TEXTE").effect("Frequence")("Curseur");

const sinus = Math.sin(time * freq * 0.1 + textIndex * 0.5);
(sinus + 1) / 2 * amp
```

> 💡 Par convention, nomme les Sliders de contrôle avec des noms explicites et regroupe-les sur un seul Null dédié au contrôle. Cela rend le projet lisible pour un autre animateur, ou pour toi-même trois semaines plus tard.

<span id="empiler"></span>
## Empiler plusieurs Expression Selectors

Un même animateur peut recevoir **plusieurs sélecteurs** (Range, Wiggly, Expression). Chacun produit une intensité, et AE les **combine** selon le mode défini dans les options du sélecteur (Ajouter, Soustraire, Intersection, Min, Max...).

**Cas d'usage : combiner une entrée séquentielle et une oscillation permanente**

- Sélecteur 1 (Range Selector) : pilote l'entrée des lettres en séquence, de gauche à droite
- Sélecteur 2 (Expression Selector) : ajoute une légère pulsation continue sur les lettres déjà visibles

```javascript
// Sélecteur 2 — oscillation légère sur les caractères déjà révélés
const sinus = Math.sin(time * 3 + textIndex * 0.8);
(sinus + 1) / 2 * 20   // amplitude faible : 0 à 20, discret sur l'opacité
```

En mode **Ajouter**, le sélecteur 2 vient s'additionner au sélecteur 1. Les lettres encore masquées restent à 0, les lettres révélées oscillent légèrement.

> 💡 L'ordre des sélecteurs dans la liste compte : AE les évalue de haut en bas. Le mode de combinaison (Ajouter, Soustraire, etc.) s'applique entre le sélecteur courant et le résultat cumulé des sélecteurs précédents.

<span id="pratique"></span>
## Cas pratique : respiration typographique

Un titre dont les lettres "respirent" en opacité et en échelle, piloté par un seul Slider "Intensite" sur un Null de contrôle.

**Structure du rig :**

```
Null "CTRL_TEXTE"
  └── Effet Curseur renommé "Intensite"  (valeur de 0 à 100)

Calque texte "TITRE"
  └── Animateur 1 : Opacité + Échelle
      └── Expression Selector
```

**Expression du sélecteur :**

```javascript
const ctrl = thisComp.layer("CTRL_TEXTE").effect("Intensite")("Curseur");

// Déphasage entre les lettres
const phase = textIndex * 0.6;

// Oscillation principale
const sinus = Math.sin(time * 1.5 + phase);

// Convertir en intensité 0-100, modulée par le Slider
(sinus + 1) / 2 * ctrl
```

**Valeurs de l'animateur :**
- Opacité : `50%` (les lettres varient entre 50% et 100% d'opacité)
- Échelle : `[95, 95]%` (légère variation de taille)

**Utilisation :**
- Slider à `0` : aucune respiration, texte statique
- Slider à `100` : respiration pleine amplitude
- Slider animé de 0 à 100 avec une courbe Ease In : la respiration démarre doucement

Ce rig est entièrement réutilisable : dupliquer le calque texte et changer le contenu suffit, l'expression reste valide.
