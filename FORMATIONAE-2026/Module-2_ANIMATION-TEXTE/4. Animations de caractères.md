<span id="proprietes"></span>
# Les propriétés animables par Animator

Dans la page précédente, on a vu comment le Text Animator fonctionne avec l'opacité. Mais un Animator peut piloter bien d'autres propriétés, et c'est là que ça devient intéressant pour le motion design.

La liste complète des propriétés disponibles dans **Animer ▼** :

```
Propriétés de transformation :
  Position          → déplacement X/Y de chaque caractère
  Ancre             → point d'ancrage de la rotation/scale
  Scale             → taille de chaque caractère
  Inclinaison       → déformation en ciseau
  Rotation          → angle de chaque caractère

Propriétés visuelles :
  Opacité           → transparence de chaque caractère
  Couleur du fond   → couleur de fond derrière chaque caractère
  Couleur de remplissage → couleur du caractère lui-même
  Couleur du contour

Propriétés typographiques :
  Tracking          → espacement entre les caractères
  Décalage de ligne → espacement entre les lignes
  Flou              → flou gaussien par caractère
```

> 💡 Toutes ces propriétés s'utilisent de la même façon : on fixe la **valeur de départ** dans l'Animator, et le Range Selector contrôle quels caractères sont affectés et dans quel ordre. La logique apprise avec l'opacité s'applique à tout.

<span id="position"></span>
# Position : faire entrer les lettres en mouvement

La Position dans un Animator définit un **décalage** par rapport à la position normale du caractère, pas une position absolue dans la composition. Une valeur de `[0, 80]` signifie "décalé de 80px vers le bas par rapport à sa position normale".

**Mise en place d'une entrée par le bas :**

1. Ajouter un Animator → Position
2. Régler la valeur à `[0, 80]` : les lettres se décalent vers le bas
3. Dans le Range Selector, animer **Fin** de 0% à 100%
4. Dans **Avancé**, régler la **Montée** à 50–80% pour une transition fluide

```
Position dans l'Animator : [0, 80]
→ les lettres hors de la plage sont décalées de 80px vers le bas
→ les lettres dans la plage reviennent à leur position normale (0, 0)
→ l'animation de Fin de 0% à 100% fait "remonter" les lettres de gauche à droite
```

**Entrée par le haut :** valeur négative sur Y → `[0, -80]`

**Entrée de côté :** jouer sur X → `[100, 0]` pour une entrée par la droite, `[-100, 0]` par la gauche.

> 💡 La valeur de Position dans l'Animator est toujours la position de **départ** (état hors sélection). La destination est toujours `[0, 0]`, la position normale du caractère dans le calque.

<span id="rotation"></span>
# Rotation : faire pivoter les caractères

La Rotation dans un Animator fait tourner chaque caractère **autour de son propre point d'ancrage**. Par défaut, ce point est au centre du caractère.

**Entrée avec rotation :**

1. Ajouter un Animator → Rotation
2. Régler la valeur à `90` ou `-90`
3. Animer le Range Selector comme d'habitude

```
Rotation : 90
→ les lettres hors de la plage sont couchées à 90°
→ elles se redressent en entrant dans la plage
```

## Modifier le point d'ancrage des caractères

Par défaut, les lettres pivotent autour de leur centre. Pour faire pivoter autour du bas de la lettre (effet de "chute" ou de "porte qui s'ouvre"), on modifie l'**Ancre** dans l'Animator :

1. Ajouter dans le même Animator → Ancre (via le bouton **Ajouter** → **Propriété**)
2. Régler l'Ancre sur `[0, 30]` environ (selon la taille de police)
   - Y positif → le pivot descend sous la lettre
   - Y négatif → le pivot monte au-dessus de la lettre

> 💡 L'Ancre dans l'Animator fonctionne différemment du point d'ancrage d'un calque classique. C'est un décalage en pixels par rapport au centre du caractère, pas une position absolue.

<span id="scale"></span>
# Scale : faire grossir ou rétrécir les lettres

La Scale dans un Animator définit la taille des caractères hors de la plage sélectionnée. Une valeur de `0%` fait apparaître les lettres en grandissant depuis zéro.

**Entrée par grossissement :**

1. Ajouter un Animator → Scale
2. Régler la valeur à `[0%, 0%]`
3. Animer le Range Selector

```
Scale : [0%, 0%]
→ les lettres hors de la plage sont invisibles (taille nulle)
→ elles grandissent jusqu'à leur taille normale en entrant dans la plage
```

**Effet de rebond :** une valeur de départ supérieure à 100%,  par exemple `[150%, 150%]` crée un effet où les lettres arrivent trop grandes et reviennent à leur taille normale. Combiné avec `ease`, ça donne un effet "pop" naturel.

## Scale X et Y indépendants

La Scale accepte des valeurs différentes sur X et Y, ce qui permet des effets d'écrasement/étirement :

```
[200%, 0%]  → les lettres s'étirent horizontalement depuis une hauteur nulle
[0%, 200%]  → les lettres s'étirent verticalement depuis une largeur nulle
```

<span id="combiner"></span>
# Combiner Position, Rotation et Scale

L'approche professionnelle consiste à combiner plusieurs propriétés **dans un même Animator** pour créer une animation cohérente et fluide. Toutes les propriétés d'un même Animator partagent le même Range Selector, elles s'appliquent donc simultanément aux mêmes caractères.

**Exemple : entrée classique "montée + fondu" :**

```
Animator 1 :
  Opacité  → 0%
  Position → [0, 60]
  Range Selector : Fin animé de 0% à 100%, Montée à 60%
```

Un seul Animator, deux propriétés, une seule animation de Range Selector, c'est la recette de base de 90% des entrées de texte en motion design.

**Exemple : entrée avec rotation et scale simultanés :**

```
Animator 1 :
  Rotation → -45°
  Scale    → [0%, 0%]
  Ancre    → [0, 20]
  Range Selector : Fin animé de 0% à 100%, Montée à 80%
```

Les lettres arrivent petites et penchées, grandissent et se redressent en même temps.

## Quand utiliser plusieurs Animators séparés

Séparer en plusieurs Animators permet de **décaler temporellement** les effets, en animant les Range Selectors avec des timings différents. Par exemple : les lettres montent (Animator 1) puis leur couleur change (Animator 2, 0,3 secondes plus tard).

<span id="opacite-flou"></span>
# Opacité et flou : les effets d'apparition

L'opacité seule donne une apparition franche. Combinée au **Flou**, elle crée un effet d'apparition progressif et doux très courant en motion design.

**Entrée avec flou :**

```
Animator 1 :
  Opacité → 0%
  Flou    → [20, 20]   (flou gaussien en pixels, X et Y)
  Range Selector : Fin animé, Montée à 70%
```

Les lettres arrivent floues et transparentes, et se mettent progressivement en netteté en entrant dans la plage.

> 💡 Le **Flou** dans un Animator est un flou gaussien appliqué individuellement à chaque caractère, il est indépendant de l'effet Flou gaussien qu'on pourrait appliquer au calque entier. C'est beaucoup plus fin et contrôlable.

**Valeurs typiques de flou pour des effets courants :**

```
[5, 5]    → flou subtil, effet de "mise au point" légère
[15, 15]  → flou visible, apparition nette et lisible
[30, 0]   → flou uniquement horizontal, effet de vitesse
[0, 30]   → flou uniquement vertical, effet de chute rapide
```

<span id="couleur"></span>
# La couleur caractère par caractère

Les Animators de couleur permettent de faire varier la teinte des caractères individuellement ou séquentiellement. Il en existe deux : **Couleur de remplissage** (la couleur du texte) et **Couleur du fond** (un rectangle de couleur derrière chaque lettre).

**Transition de couleur sur l'ensemble du mot :**

1. Ajouter un Animator → Couleur de remplissage
2. Choisir la couleur de départ dans l'Animator
3. Animer le Range Selector pour que la couleur "balaie" le mot de gauche à droite

```
Couleur de remplissage : rouge
→ les lettres dans la plage sont rouges
→ les lettres hors de la plage gardent leur couleur d'origine
→ la Montée à 0% crée une transition lettre par lettre nette
→ la Montée à 100% crée un dégradé progressif sur tout le mot
```

**Effet de surlignage animé avec la Couleur du fond :**

```
Animator :
  Couleur du fond    → jaune (couleur du surlignage)
  Scale              → [100%, 110%]  (légèrement plus grand que la lettre)
  Range Selector     → Fin animé de 0% à 100%, Montée à 0%
```

Le surlignage passe lettre par lettre de gauche à droite.

<span id="animation-complete"></span>
# Construire une animation complète

Voici la démarche complète pour construire une animation de texte professionnelle, de A à Z.

**Objectif : un titre qui entre mot par mot, avec montée, fondu et légère rotation.**

**Étape 1 : préparer le calque**

Créer le calque texte. S'assurer que le point d'ancrage du calque est bien positionné (centre ou bas du texte selon l'effet voulu).

**Étape 2 : créer l'Animator principal**

Animer ▼ → Opacité. Régler à `0%`. Puis dans ce même Animator, ajouter via **Ajouter → Propriété** : Position `[0, 40]` et Rotation `-10°`.

**Étape 3 : configurer le Range Selector**

Dans le Range Selector → **Avancé** :
- Basé sur : **Mots** (les mots entrent un par un, pas les lettres)
- Montée : `60%`

**Étape 4 : animer le Range Selector**

À `0s` : Fin = `0%`. À `1s` : Fin = `100%`. Ajouter une interpolation `easeOut` sur les keyframes pour une entrée qui freine naturellement.

**Étape 5 : ajuster le timing**

Déplacer les keyframes pour que l'animation dure ce qu'il faut. Jouer sur la Montée pour adoucir ou durcir les transitions entre mots.

**Résultat :**

```
Animator 1 :
  Opacité  → 0%
  Position → [0, 40]
  Rotation → -10°
  Range Selector :
    Basé sur   : Mots
    Montée     : 60%
    Fin        : 0% → 100% sur 1 seconde (easeOut)
```

> 💡 La majorité des animations de texte professionnelles ne dépassent pas 2 Animators. La complexité vient de la combinaison des propriétés et du réglage fin de la Montée, pas de l'accumulation d'effets.
