## C'est quoi un Text Animator ?

Un Text Animator est un système d'animation **propre aux calques de texte** dans AE. Il permet d'appliquer une transformation : position, rotation, opacité, échelle, couleur...  **caractère par caractère**, mot par mot, ou ligne par ligne, sans keyframe sur chaque élément individuellement.

> 💡 Analogie : imagine que tu veuilles faire entrer chaque lettre d'un mot l'une après l'autre. Sans Animator, il faudrait un calque par lettre, et autant de keyframes. Avec un Animator, un seul calque de texte suffit, tu définis la transformation et AE la distribue sur les caractères automatiquement.

Ce qui rend les Animators puissants, c'est la séparation entre deux éléments distincts :
- le **Animator** définit *quoi* animer (opacité, position, rotation…)
- le **Selector** définit *qui* est affecté (quels caractères, dans quel ordre, avec quelle intensité)

<span id="animator"></span>
## Ajouter un Animator

Dans la timeline, déplie les propriétés de ton calque texte → **Texte** → clique sur le bouton **Animer** (à droite du mot "Texte"). Un menu apparaît avec toutes les propriétés animables par Animator.

```
Calque texte
└── Texte
    ├── Animator 1
    │   ├── Range Selector 1    ← contrôle qui est affecté
    │   └── Opacité             ← la propriété animée (valeur = 0%)
    └── [Animer ▼]              ← bouton pour ajouter un Animator
```

Plusieurs propriétés peuvent être ajoutées dans un même Animator, et plusieurs Animators peuvent coexister sur le même calque, ils s'appliquent dans l'ordre, de haut en bas.

## Le Range Selector

Le Range Selector est le sélecteur par défaut. Il définit une **plage** de caractères affectés par l'Animator, avec un début et une fin.

**Les paramètres essentiels :**

```
Début    → position de départ de la plage (0% = premier caractère)
Fin      → position de fin de la plage (100% = dernier caractère)
Décalage → déplace la plage sans changer sa taille
```

**L'animation type : un mot qui entre lettre par lettre :**

1. Ajouter un Animator → Opacité
2. Régler la valeur d'Opacité dans l'Animator à **0%**
3. Dans le Range Selector, animer **Fin** de 0% à 100% sur la durée souhaitée

AE fait entrer les lettres de gauche à droite, la plage "sélectionnée" a une opacité de 0%, tout ce qui est hors de la plage garde son opacité normale (100%).

**La section Avancé du Range Selector** contrôle comment la valeur s'applique aux caractères à la frontière de la plage :

```
Unités    → Pourcentage ou Index (caractère précis)
Basé sur  → Caractères / Mots / Lignes
Mode      → Ajouter, Soustraire, Intersect…
Montée    → la zone de transition entre sélectionné et non-sélectionné
              0% = transition franche / 100% = transition progressive sur toute la plage
```

> 💡 Le paramètre **Montée** est clé pour un effet fluide. À 0%, chaque lettre apparaît d'un coup, effet mécanique. À 100%, les lettres se fondent progressivement les unes dans les autres, effet organique.

<span id="wiggly"></span>
## Le Wiggly Selector

Le Wiggly Selector remplace ou s'ajoute au Range Selector. Au lieu d'une plage définie, il affecte les caractères de manière **aléatoire et animée dans le temps**, chaque lettre reçoit une intensité différente, qui fluctue en continu.

**Pour l'ajouter :** dans l'Animator, clique sur **Ajouter** → **Sélecteur** → **Agitation**.

**Les paramètres clés :**

```
Mode           → comment il interagit avec les autres sélecteurs (Ajouter, Intersect…)
Intensité max  → intensité maximale appliquée à un caractère (100% = effet plein)
Intensité min  → intensité minimale (0% = caractère non affecté)
Basé sur       → Caractères / Mots / Lignes
Fréquence      → vitesse d'oscillation (oscillations par seconde)
Corrélation    → 0% = chaque caractère indépendant / 100% = tous évoluent ensemble
Verrou         → même résultat à chaque rendu (équivalent du seedRandom)
```

> 💡 Combiné à un Animator Rotation ou Position, le Wiggly Selector crée un effet de texte "vivant" en quelques secondes, sans une seule keyframe.

<span id="combiner"></span>
## Combiner plusieurs Animators

Sur un même calque texte, plusieurs Animators se cumulent. L'ordre dans la timeline détermine l'ordre d'application.

**Exemple : entrée animée avec opacité ET position :**

```
Animator 1 : Opacité → 0%
  Range Selector : Fin animé de 0% à 100%

Animator 2 : Position → [0, 50]  (décalage vertical de 50px)
  Range Selector : même animation que l'Animator 1
```

Les deux effets s'appliquent simultanément sur les mêmes caractères, les lettres montent et apparaissent en même temps. C'est la base de la majorité des animations de texte professionnelles.
